Rotating Vectors

Let

{% highlight matlab %} o = rotation.byEuler(90*degree,90*degree,0*degree) {% endhighlight %} {% highlight plaintext %} o = rotation Bunge Euler angles in degree phi1 Phi phi2 90 90 0 {% endhighlight %}

a certain rotation. Then the rotation of the xvector is computed via

{% highlight matlab %} v = o * xvector {% endhighlight %} {% highlight plaintext %} v = vector3d (y↑→x) x y z 0 1 0 {% endhighlight %}

The inverse rotation is computed via the backslash operator

{% highlight matlab %} o \ v {% endhighlight %} {% highlight plaintext %} ans = vector3d (y↑→x) x y z 1 0 0 {% endhighlight %}

Concatenating Rotations

Let

{% highlight matlab %} rot1 = rotation.byEuler(90*degree,0,0); rot2 = rotation.byEuler(0,60*degree,0); {% endhighlight %}

be two rotations. Then the rotation defined by applying first rotation one and then rotation two is computed by

{% highlight matlab %} rot = rot2 * rot1 {% endhighlight %} {% highlight plaintext %} rot = rotation Bunge Euler angles in degree phi1 Phi phi2 0 60 90 {% endhighlight %}

Rotational angle and the rotational axis

Then rotational angle and the axis of rotation can be computed via then commands angle(rot) and axis(rot)

{% highlight matlab %} rot.angle / degree rot.axis {% endhighlight %} {% highlight plaintext %} ans = 104.48 ans = vector3d (y↑→x) x y z 0.447214 -0.447214 0.774597 {% endhighlight %}

If two rotations are specifies the command angle(rot1,rot2) computes the rotational angle between both rotations

{% highlight matlab %} angle(rot1,rot2) / degree {% endhighlight %} {% highlight plaintext %} ans = 104.48 {% endhighlight %}

The inverse Rotation

The inverse rotation you get from the command inv(rot)

{% highlight matlab %} inv(rot) {% endhighlight %} {% highlight plaintext %} ans = rotation Bunge Euler angles in degree phi1 Phi phi2 90 60 180 {% endhighlight %}

Conversion into Euler Angles and Rodrigues Parametrisation

There are methods to transform rotations in almost any other parameterization of rotations as they are:

  • Euler(rot) in Euler angle
  • Rodrigues(rot) in Rodrigues parameter
{% highlight matlab %} [alpha,beta,gamma] = Euler(rot,'Matthies') {% endhighlight %} {% highlight plaintext %} alpha = 4.7124 beta = 1.0472 gamma = 3.1416 {% endhighlight %}
