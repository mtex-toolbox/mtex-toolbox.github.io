Sections of the orientations space

A third possibility are two dimensional sections through the Euler angle space. The most popular type of such sections are the so called phi2 sections.

{% highlight matlab %} cs = crystalSymmetry('432') ss = specimenSymmetry('222') ori = orientation.rand(100,cs,ss) % as phi2 sections plotSection(ori,'phi2') {% endhighlight %} {% highlight plaintext %} cs = crystalSymmetry symmetry: 432 elements: 24 a, b, c : 1, 1, 1 ss = orthorhombic specimenSymmetry ori = orientation (432 → y↑→x (222)) size: 100 x 1 {% endhighlight %}
