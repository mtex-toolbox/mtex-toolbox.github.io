Theory

The Bingham distribution has the density function

\[ f(g;K,U) = _1\!F_1 \left(\frac{1}{2},2,K \right)^{-1} \exp \left\{ g^T UKU g \right\},\qquad g\in S^3, \]

where \(U\) is an \(4 \times 4\) orthogonal matrix with unit quaternions \(u_{1,..,4}\in S^3\) in the columns and \(K\) is a \(4 \times 4\) diagonal matrix with the entries \(k_1,..,k_4\) describing the shape of the distribution. \(_1F_1(\cdot,\cdot,\cdot)\) is the hypergeometric function with matrix argument normalizing the density.

The shape parameters \(k_1 \ge k_2 \ge k_3 \ge k_4\) give

  • a bipolar distribution, if \(k_1 + k_4 > k_2 + k_3\),
  • a circular distribution, if \(k_1 + k_4 = k_2 + k_3\),
  • a spherical distribution, if \(k_1 + k_4 < k_2 + k_3\),
  • a uniform distribution, if \(k_1 = k_2 = k_3 = k_4\),

The general setup of the Bingham distribution in MTEX is done as follows

{% highlight matlab %} cs = crystalSymmetry('1'); kappa = [100 90 80 0]; % shape parameters U = eye(4); % orthogonal matrix odf = BinghamODF(kappa,U,cs) {% endhighlight %} {% highlight plaintext %} odf = SO3FunBingham (1 → y↑→x) kappa: 100 90 80 0 weight: 1 {% endhighlight %}

Lets visualize the ODF as pole figures

{% highlight matlab %} h = Miller({0,0,1},{1,0,0},{1,1,1},cs); plotPDF(odf,h,'antipodal','silent','layout',[1 3]); {% endhighlight %}

and in Euler angle space

Estimating the parameters of a Bingham distribution

The importance of the Bingham distribution is that it is a quite low dimensional model for an orientation distribution function that still is flexible enough to represent different kinds of textures like fibers and unimodal distributions. Furthermore, we may estimate Bingham distribution from a set of individual orientations, coming e.g. from an EBSD measurement or a plasticity simulation. In contrast to kernel density estimation estimating the parameters of the Bingham distributions requires much less data. Lets demonstrate the process of fitting a Bingham ODF to experimental data. To this end we start with a randomly aligned fibre ODF

{% highlight matlab %} odfTrue = fibreODF(fibre.rand(cs)); plotPDF(odfTrue,h,'antipodal','silent') {% endhighlight %}
{% include inline_image.html file="BinghamODFs_03.png" %}

Next we use this fibre ODF to simulate only 1000 random orientations using the command discreteSample

{% highlight matlab %} ori = discreteSample(odfTrue,1000); plot(ori,'add2all','MarkerEdgeColor','k',... 'MarkerSize',5,'MarkerFaceColor','none','MarkerEdgeAlpha',0.2) {% endhighlight %}
{% include inline_image.html file="BinghamODFs_04.png" %}

To those simulated orientation data we can now fit a Bingham distribution using the command calcBinghamODF

{% highlight matlab %} odf = calcBinghamODF(ori) plotPDF(odf,h,'antipodal','silent') {% endhighlight %} {% highlight plaintext %} odf = SO3FunBingham (1 → y↑→x) kappa: 0 28 198 198 weight: 1 {% endhighlight %}
{% include inline_image.html file="BinghamODFs_05.png" %}

We observe an almost perfect fit between the original fibre ODF and the Bingham distribution estimated from only 1000 randomly drawn orientations.

Specific Bingham distributions

In the following we present the three corner cases of the Bingham distribution: the unimodal distribution, the fibre distribution, and the spherical distribution.

The unimodal case A unimodal Bingham distribution with reference orientation oriRef and kappa=40 is constructed by

{% highlight matlab %} % a modal orientation cs = crystalSymmetry('321'); oriRef = orientation.byEuler(45*degree,0*degree,0*degree,cs); % the corresponding Bingham ODF odf = BinghamODF(20,oriRef) plot(odf,'sections',6,'silent','contourf','sigma') {% endhighlight %} {% highlight plaintext %} odf = SO3FunBingham (321 → y↑→x) kappa: 20 0 0 0 weight: 1 {% endhighlight %}
{% include inline_image.html file="BinghamODFs_06.png" %}

The fibre case For a fibre symmetric Bingham distribution we simply specify the fibre and the first kappa parameter. The first two kappa parameters are allways equal while the third and fourth are zero.

{% highlight matlab %} f = fibre.rand(cs); odf = BinghamODF(20,f) plot(odf,'sections',6,'silent','sigma') {% endhighlight %} {% highlight plaintext %} odf = SO3FunBingham (321 → y↑→x) kappa: 20 20 0 0 weight: 1 {% endhighlight %}
{% include inline_image.html file="BinghamODFs_07.png" %}

The spherical case The spherical case is characterized by the fact that we have 3 equal non zero kappa coefficients.

{% highlight matlab %} odf = BinghamODF([10,10,10],quaternion(eye(4)),cs) plot(odf,'sections',6,'silent','sigma'); {% endhighlight %} {% highlight plaintext %} odf = SO3FunBingham (321 → y↑→x) kappa: 10 10 10 0 weight: 1 {% endhighlight %}
{% include inline_image.html file="BinghamODFs_08.png" %}

Prolate case and fiber distribution

The prolate case corresponds to a fiber.

{% highlight matlab %} %odf_prolate = fibreODF(fibre.rand(cs),'halfwidth',20*degree) %plotPDF(odf_prolate,h,'upper','silent') {% endhighlight %}

As before, we generate some random orientations from a model odf. The shape in an axis/angle scatter plot reminds of a cigar

{% highlight matlab %} %ori_prolate = discreteSample(odf_prolate,10000); %plot(ori_prolate,'axisAngle') {% endhighlight %}

We estimate the parameters of the Bingham distribution

{% highlight matlab %} %odf = calcBinghamODF(ori_prolate) %plotPDF(odf,h,'upper','silent') {% endhighlight %}

and test on the three cases

{% highlight matlab %} %T_spherical = bingham_test(ori_prolate,'spherical','approximated'); %T_oblate = bingham_test(ori_prolate,'prolate', 'approximated'); %T_prolate = bingham_test(ori_prolate,'oblate', 'approximated'); %t = [T_spherical T_oblate T_prolate] {% endhighlight %}

The test clearly rejects the spherical and prolate case, but not the prolate. We construct the Bingham distribution from the parameters, it might show some skewness

{% highlight matlab %} %odf_prolate = BinghamODF(kappa,U,cs) %plotPDF(odf_prolate,h,'antipodal','silent') {% endhighlight %}
