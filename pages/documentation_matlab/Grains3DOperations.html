On this page we explain some basic operations with three dimensional grains. Let us start by importing some example data set and plot it from a nice perspective

{% highlight matlab %} mtexdata NeperGrain3d % colorize by mean orientation plot(grains,grains.meanOrientation) setCamera(plottingConvention.default3D) {% endhighlight %} {% highlight plaintext %} grains = grain3d Phase Grains Volume Mineral Symmetry Crystal reference frame 2 1000 1000000 Quartz 321 X||a*, Y||b, Z||c* boundary faces: 7203 Properties: meanRotation {% endhighlight %}

Slicing

We can extract from 3d grain data 2d grain data by slicing them along one or multiple planes. This is done using the command slice. This command requires two inputs to characterize a plane - the plane normal N and an arbitrary point P0 within the plane.

{% highlight matlab %} % a point where the slice should pass through P0 = vector3d(50,50,50); % the normal direction of the slice N = vector3d(1,-1,1); % compute the slice grains1_10 = grains.slice(N,P0) % visualize the slice plot(grains1_10,grains1_10.meanOrientation,'micronbar','off') setCamera(plottingConvention.default3D) {% endhighlight %} {% highlight plaintext %} grains1_10 = grain2d (y↑→x) Phase Grains Pixels Mineral Symmetry Crystal reference frame 2 183 183 Quartz 321 X||a*, Y||b, Z||c* boundary segments: 552 (3170 µm) inner boundary segments: 0 (0 µm) triple points: 320 Properties: meanRotation, Id3d {% endhighlight %}
{% include inline_image.html file="Grains3DOperations_02.png" %}

We may adjust the plottingConvention such that the normal direction is perpendicular to the screen.

{% highlight matlab %} how2plot = plottingConvention; how2plot.outOfScreen = N; how2plot.north = zvector setCamera(how2plot) {% endhighlight %} {% highlight plaintext %} how2plot = plottingConvention (xyz) outOfScreen: (4,-7,0) north : (0,0,1) east : (7,4,0) {% endhighlight %}
{% include inline_image.html file="Grains3DOperations_03.png" %}

We may use the exact same syntax to generate multiple slices.

{% highlight matlab %} N = vector3d.Z; for k = 1:19:99 grainSlice = grains.slice(N, vector3d(0,0,k)); plot(grainSlice,grainSlice.meanOrientation) hold on end hold off setCamera(plottingConvention.default3D) {% endhighlight %}
{% include inline_image.html file="Grains3DOperations_04.png" %}

Triangulation

Some functions are much faster on triangulated meshes. Therefore you can triangulate your grains with the command triangulate.

{% highlight matlab %} grainsTri = grains(20:21).triangulate plot(grainsTri,grainsTri.meanOrientation) {% endhighlight %} {% highlight plaintext %} grainsTri = grain3d Phase Grains Volume Mineral Symmetry Crystal reference frame 2 2 2149 Quartz 321 X||a*, Y||b, Z||c* boundary faces: 112 Id Phase Pixels meanRotation 20 2 1 (125.7°,29.9°,207.1°) 21 2 1 (356.2°,22.8°,16.4°) {% endhighlight %}
{% include inline_image.html file="Grains3DOperations_05.png" %}

Rotation

Not surprisingly we can use the command rotate to apply any rotation to three dimensional grains. Note that a rotation changes the spatial coordinates as well as the orientation of the grains.

{% highlight matlab %} rot = rotation.byAxisAngle(vector3d(1,1,1),30*degree); grains_rot = rot * grains; % or rotate(grains3,rot) % plotting plot(grains_rot,grains_rot.meanOrientation) {% endhighlight %}
