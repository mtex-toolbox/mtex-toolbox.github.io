Inverse pole figures are two dimensional representations of orientations. To illustrate this we define a random orientation with trigonal crystal symmetry

{% highlight matlab %} cs = crystalSymmetry('321'); ori = orientation.rand(cs) {% endhighlight %} {% highlight plaintext %} ori = orientation (321 → y↑→x) Bunge Euler angles in degree phi1 Phi phi2 156.958 161.468 197.878 {% endhighlight %}

Starting point is a fixed specimen direction r, e.g.,

{% highlight matlab %} r = vector3d.Z; {% endhighlight %}

Next the crystal direction that aligns with specimen direction r according to the orientation ori is computed and plotted in a spherical projection

{% highlight matlab %} h = inv(ori) * r plot(h.symmetrise,'fundamentalRegion') {% endhighlight %} {% highlight plaintext %} h = Miller (321) h k i l 0.0667 -0.3025 0.2357 -0.9481 {% endhighlight %}
{% include inline_image.html file="OrientationInversePoleFigure_01.png" %}

A shortcut for the above computations is the command plotIPDF which takes as second argument an arbitrary list of specimen directions r

{% highlight matlab %} % a pole figure plot plotIPDF(ori,[vector3d.X,vector3d.Y,vector3d.Z]) {% endhighlight %}
{% include inline_image.html file="OrientationInversePoleFigure_02.png" %}

Contour plots

Using the option 'contourf' we may turn those scatter plots into contour plots.

{% highlight matlab %} plotIPDF(ori,[vector3d.X,vector3d.Y,vector3d.Z],'contourf') mtexColorbar {% endhighlight %}
{% include inline_image.html file="OrientationInversePoleFigure_03.png" %}

Note that by default only the fundamental sector with respect to the crystal symmetry is plotted. In order to plot the entire sphere use the option 'complete'.

{% highlight matlab %} plotIPDF(ori,[vector3d.X,vector3d.Y,vector3d.Z],'contourf','complete','upper') mtexColorbar {% endhighlight %}
