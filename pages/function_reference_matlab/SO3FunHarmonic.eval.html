evaluates the rotational harmonic on a given set of points using NFSOFT

gives a align for numel(F)==1 or otherwise transform the align of v to a vector.

Syntax

{% highlight matlab %} f = eval(F,v) {% endhighlight %}

Input

F SO3FunHarmonic
v rotation interpolation nodes

Output

f double
