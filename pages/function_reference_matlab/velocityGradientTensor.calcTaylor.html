compute Taylor factor and strain dependent orientation gradient

Syntax

{% highlight matlab %} [M,b,W] = calcTaylor(eps,sS) {% endhighlight %}

Input

L velocityGradientTensor
sS slipSystem

Output

M taylor factor
b coefficients for the acive slip systems
W spinTensor

Example

consider uniaxial tension in (100) direction about 30 percent

{% highlight matlab %} F = deformationGradientTensor.uniaxial(vector3d.X,1.3) {% endhighlight %} {% highlight plaintext %} F = deformationGradientTensor (1) rank: 2 (3 x 3) 1.3 0 0 0 0.8771 0 0 0 0.8771 {% endhighlight %}

the corresponding rate of deformation tensor becomes

{% highlight matlab %} L = logm(F) {% endhighlight %} {% highlight plaintext %} L = velocityGradientTensor (1) rank: 2 (3 x 3) *10^-2 26.236 0 0 0 -13.118 0 0 0 -13.118 {% endhighlight %}

define a crystal orientation

{% highlight matlab %} cs = crystalSymmetry('cubic'); ori = orientation.byEuler(0,30*degree,15*degree,cs); {% endhighlight %}

define a slip system

{% highlight matlab %} sS = slipSystem.fcc(cs) {% endhighlight %} {% highlight plaintext %} sS = slipSystem (m-3m) u v w | h k l CRSS 0 1 -1 1 1 1 1 {% endhighlight %}

compute the Taylor factor

{% highlight matlab %} [M,b,W] = calcTaylor(inv(ori)*L,sS.symmetrise); {% endhighlight %}

update orientation

{% highlight matlab %} oriNew = ori .* orientation(-W) {% endhighlight %} {% highlight plaintext %} oriNew = orientation (m-3m → y↑→x) Bunge Euler angles in degree phi1 Phi phi2 1.54702e-10 30 11.2419 {% endhighlight %}
