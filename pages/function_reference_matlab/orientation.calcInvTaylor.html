Taylor factor from orientation gradient

Syntax

{% highlight matlab %} [M,b,eps] = calcInvTaylor(mori,sS) {% endhighlight %}

Input

mori misorientation
sS slipSystem list in crystal coordinates

Output

M taylor factor
b coefficients for the acive slip systems
eps strain tensor list in crystal coordinates

Example

define 10 percent strain

{% highlight matlab %} eps = 0.1 * strainTensor(diag([1 -0.75 -0.25])) {% endhighlight %} {% highlight plaintext %} eps = strainTensor (y↑→x) type: Lagrange rank: 2 (3 x 3) *10^-2 10 0 0 0 -7.5 0 0 0 -2.5 {% endhighlight %}

define a crystal orientation

{% highlight matlab %} cs = crystalSymmetry('cubic') ori = orientation.byEuler(0,30*degree,15*degree,cs) {% endhighlight %} {% highlight plaintext %} cs = crystalSymmetry symmetry: m-3m elements: 48 a, b, c : 1, 1, 1 ori = orientation (m-3m → y↑→x) Bunge Euler angles in degree phi1 Phi phi2 0 30 15 {% endhighlight %}

define a slip system

{% highlight matlab %} sS = slipSystem.fcc(cs) {% endhighlight %} {% highlight plaintext %} sS = slipSystem (m-3m) u v w | h k l CRSS 0 1 -1 1 1 1 1 {% endhighlight %}

compute the Taylor factor
