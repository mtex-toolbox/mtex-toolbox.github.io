return index of all points in an epsilon neighborhood of a vector

Syntax

{% highlight matlab %} [ind,d] = find(v,w) % find closest point out of v to w [I,d] = find(v,w,epsilon) % find all points out of v in an epsilon neighborhood of w [ind,d] = find(v,w,k) % find k nearest points out of v to w {% endhighlight %}

Input

v, w vector3d
epsilon double
k int32

Output

ind int32 array for k nearest neighbors,
I sparse logical incidence matrix for region search
d distance to the found neighbors, same size as ind or I

Options

antipodal include antipodal symmetry
