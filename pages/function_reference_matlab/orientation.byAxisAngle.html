define orientations by rotational axis and rotational angle

Syntax

{% highlight matlab %} ori = orientation.byAxisAngle(v,omega,CS) % an orientation ori = orientation.byAxisAngle(h,omega) % a misorientation {% endhighlight %}

Input

v rotational axis vector3d
m rotational axis Miller
omega rotation angle
CS crystalSymmetry

Output

ori orientation
