random directions or axes

Syntax

{% highlight matlab %} v = rand % a random unit vector v = rand('antipodal') % a random axis v = rand(m,n) % a m x n matric of random unit vectors {% endhighlight %} {% highlight matlab %} cs = crystlSymmetry('432'); sR = cs.fundamentalSector; v = rand(v,sR) % a random unit vector inside a spherical region {% endhighlight %}

Input

m,n double
sR sphericalRegion

Output

v vector3d
