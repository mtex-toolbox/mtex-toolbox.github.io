load pole figure data from (alpha,beta,gamma) files

Description

loadVector3d_generic reads vector3d from any txt or exel files. The assoziation of the columns as cartesian coordinates or polar angles is specified by the options ColumnNames and Columns. The files can be contain any number of header lines.

Syntax

{% highlight matlab %} v = loadVector3d_generic(fname,'ColumnNames',{'x','y','z'}) v = loadVector3d_generic(fname,'ColumnNames',{'latitude','longitude'}) v = loadVector3d_generic(fname,'ColumnNames',{'polar angle','azimuth'}) {% endhighlight %}

Input

fname file name (text files only)

Options

ColumnNames names of the colums to be imported
Columns postions of the columns to be imported
RADIANS treat input in radiand
DELIMITER delimiter between numbers

See also
