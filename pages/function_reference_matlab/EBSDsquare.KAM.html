intragranular average misorientation angle per orientation

Syntax

{% highlight matlab %} plot(ebsd,ebsd.KAM ./ degree) {% endhighlight %} {% highlight matlab %} % ignore misorientation angles > threshold kam = KAM(ebsd,'threshold',10*degree); plot(ebsd,kam./degree) {% endhighlight %} {% highlight matlab %} % ignore grain boundary misorientations [grains, ebsd.grainId] = calcGrains(ebsd) plot(ebsd, ebsd.KAM./degree) {% endhighlight %} {% highlight matlab %} % consider also second order neighbors kam = KAM(ebsd,'order',2); plot(ebsd,kam./degree) {% endhighlight %}

Input

ebsd EBSD

Options

threshold ignore misorientation angles larger then threshold
order consider neighbors of order n
max take not the mean but the maximum misorientation angle
weights filter mask
