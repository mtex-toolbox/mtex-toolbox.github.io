import orientation data from data files

Description

rotation.load is a high level method for importing rotations from column aligned text files or excel spread sheets. In those cases it is neccesary to tell MTEX the column positions of the spatial coordinates, the phase information as well as Euler angles.

Syntax

{% highlight matlab %} rot = rotation.load(fname,'ColumnNames',{'phi1','Phi','phi2'}) {% endhighlight %}

Input

fname filename

Output

ori orientation

Options

columnNames names of the colums to be imported, mandatory are euler 1, euler 2, euler 3
columns postions of the columns to be imported
radians treat input in radiand
delimiter delimiter between numbers
header number of header lines
Bunge [phi1 Phi phi2] Euler angle in Bunge convention (default)
passive
