compute the Schmid factor

Syntax

{% highlight matlab %} SFfun = SchmidFactor(sS) % returns spherical function SF = SchmidFactor(sS,v) SF = SchmidFactor(sS,sigma) SF = SchmidFactor(sS,sigma,'relative') {% endhighlight %}

Input

sS list of slipSystem
v vector3d list of tension direction
sigma stressTensor

Output

SFfun size(sS) x 1 list of S2FunHarmonic
SF size(sS) x size(sigma) matrix of Schmid factors
