Recursively assigns points based on neighboring assignments If flag == 0, the point is fresh, 1, the point is set >1, the point belongs to a cluster that may have run into the recusion limit

Syntax

{% highlight matlab %} [assignmentsN, flag, contPoints] = clusterBreaker(assignmentsN, point, assignments, flag, NNlist, oldClust, newClust, contPoints, rDepth, rLimit) {% endhighlight %}
