a class representing a harmonic function on the rotational group.

Syntax

{% highlight matlab %} SO3F = SO3FunHarmonic(fhat) SO3F = SO3FunHarmonic(fhat,CS,SS) SO3F = SO3FunHarmonic(F) {% endhighlight %}

Input

fhat double (harmonic coefficient vector)
CS,SS @Symmetry
F SO3Fun

Output

SO3F SO3FunHarmonic

Options

bandwidth bandwidth of the harmonic expansion

Example

{% highlight matlab %} F = SO3FunHandle(@(rot) rot.angle) {% endhighlight %} {% highlight plaintext %} F = SO3FunHandle (y↑→x → y↑→x) eval: @(rot)rot.angle {% endhighlight %}
