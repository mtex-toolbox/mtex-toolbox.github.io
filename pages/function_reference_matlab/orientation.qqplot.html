quantile-quantile of misorientation angle against random angular misorientation

Syntax

{% highlight matlab %} qqplot(o) {% endhighlight %}

Example

{% highlight matlab %} cs = crystalSymmetry('-43m'); odf1 = unimodalODF(orientation.id(cs),'halfwidth',20*degree); odf2 = unimodalODF(orientation.id(cs),'halfwidth',50*degree); {% endhighlight %} {% highlight matlab %} qqplot(odf1.discreteSample(1000)) qqplot(odf2.discreteSample(1000)) {% endhighlight %}
