rotate grains

Syntax

{% highlight matlab %} % rotate the grains about the z-axis by 90*degree grains = rotate(grains,10*degree) {% endhighlight %} {% highlight matlab %} % rotate about the x-axis grains = rotate(grains,rotation.byAxisAngle(xvector,180*degree)) {% endhighlight %} {% highlight matlab %} % rotate about a specific point grains = rotate(grains,180*degree,'center',[0,0]) {% endhighlight %}

Input

grains grain2d
angle double
rot rotation

Output

grains grain2d

Options

center [x,y] center of rotation, default is (0,0)

Flags

keepXY rotate only the orientation data, i.e. the Euler angles
keepEuler rotate only the spatial data, i.e., the x,y, and z values
