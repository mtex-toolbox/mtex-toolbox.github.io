returns the Ids of neighboring grains

Syntax

{% highlight matlab %} % pairs of neighboring grains in grains1 pairs = neighbors(grains1) {% endhighlight %} {% highlight matlab %} % pairs of neighboring grains in grains1 and grains2 pairs = neighbors(grains1,grains2) {% endhighlight %} {% highlight matlab %} % all neighboring grains of grains1 pairs = neighbors(grains1,'full') {% endhighlight %} {% highlight matlab %} % adjacency matrix of grains A = grains.neighbors('matrix') {% endhighlight %}

Input

grains1, grains2 grain2d

Output

pairs index list of size N x 2
A adjacency matrix

Options

full consider all neighboring grains, not only those contained in
matrix provide the result as adjacency matrix
