projects quaternions to a fundamental region

Syntax

{% highlight matlab %} project2FundamentalRegion(q,CS) % to FR project2FundamentalRegion(q,CS,q_ref) % to FR around reference rotation project2FundamentalRegion(q,CS1,CS2,q_ref) % misorientation to FR around id {% endhighlight %}

Input

q quaternion
CS1, CS2 crystal symmetry
q_ref reference quaternion single or size(q) == size(q_ref)

Output

q quaternion
omega rotational angle to reference quaternion
