--- title: finiteStrain sidebar: function_reference_sidebar permalink: velocityGradientTensor.finiteStrain.html folder: function_reference toc: false --- finiteStrain edit page

derive finte strain axes and magnitudes from deformation tensor after n-steps (in strain rate units of L) using the solution of Provost et al.2014 doi:10.1029/2001JB001734

Syntax

{% highlight matlab %} [fe, qe, Ea] = finiteStrain(L,n) {% endhighlight %}

Input

L velocityGradientTensor
n time step in units of strainrate*2 of L

Output

fe finite strain ellipsoid axis directions (vector3d)
qe length of ellipse axes
Ea finite strain tensor (Langrange)
