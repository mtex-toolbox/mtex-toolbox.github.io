decomposes the tensor into

Syntax

{% highlight matlab %} % decompose into symmetric portions [Tiso, T1, T2, T3] = symmetricDecomposition(T,cs1,cs2,cs3) {% endhighlight %}

cs1, cs2, cs3 symmetry

Output
