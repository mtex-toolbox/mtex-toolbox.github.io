get the maximum angle of the fundamental region

Description

It is important that v is within the fundamental sector

Syntax

{% highlight matlab %} omega = maxAngle(oR) % maximum angle in the orientation region omega = maxAngle(oR,v) % maximum angle about axis v {% endhighlight %}

Input

oR orientationRegion
v vector3d
