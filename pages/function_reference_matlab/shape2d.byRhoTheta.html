define shape2d by polar angles

Syntax

{% highlight matlab %} v = shape2d.byRhoTheta(rho,theta) {% endhighlight %}

Input

rho radius
theta angle to the x axis when projected into the x/y plane in radiant

Output

shape shape2d

Example

{% highlight matlab %} mtexdata forsterite silent grains = calcGrains(ebsd('indexed')); omega = [0:360]*degree; cumpf = surfor(grains.boundary('f','f'),omega); shape = shape2d.byRhoTheta(cumpf, omega) plot(shape) {% endhighlight %} {% highlight plaintext %} shape = shape2d vertices: 361 {% endhighlight %}
