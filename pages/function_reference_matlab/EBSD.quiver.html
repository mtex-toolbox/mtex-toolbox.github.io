plot directions at ebsd centers

Syntax

{% highlight matlab %} quiver(ebsd,dir,'linecolor','r') {% endhighlight %} {% highlight matlab %} mtexdata small quiver(ebsd,ebsd.rotations.axis) {% endhighlight %}

Input

ebsd EBSD
dir vector3d

Options

antipodal plot directions or axes
maxHeadSize size of the arrow
project2Plane project direction into plane
