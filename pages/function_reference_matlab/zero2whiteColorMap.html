Define a color map which strongly depends on the maximal and minimal value of the plot. The colormap plots all positive values with the WhiteJetColorMap. The negative values are plotted as magenta.

Syntax

{% highlight matlab %} map = zero2whiteColorMap(CLim,n) {% endhighlight %}
