function to fit Bingham parameters

Description

confidence ellipse for the mean direction based on Tanaka (1999) https://doi.org/10.1186/BF03351601

Syntax

{% highlight matlab %} BS2 = BinghamS2.fit(v) {% endhighlight %}

Input

v vector3d

Output

BS2 BinghamS2

Options

ConfElli confidence level p (default at 0.95)

Example

{% highlight matlab %} % simulate some directions odf = unimodalODF(quaternion.id,'halfwidth',10*degree); N = 100; v = odf.discreteSample(N) .* ... rotation.byAxisAngle(vector3d.X,rand(N,1)*2*pi) * vector3d.Y; {% endhighlight %} {% highlight matlab %} % fit a Bingham distribution S2F = BinghamS2.fit(v) {% endhighlight %} {% highlight plaintext %} S2F = BinghamS2 (y↑→x) {% endhighlight %}
