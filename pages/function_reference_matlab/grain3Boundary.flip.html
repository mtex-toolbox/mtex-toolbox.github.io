flip a boundary from phase1/phase2 to phase2/phase1

Syntax

{% highlight matlab %} gB3 = flip(gB3) % flip all boundaries gB3 = flip(gB3,ind) % flip the boundaries specified in ind {% endhighlight %}

Input

gB grain3Boundary
ind indices of the boundaries to flip

Output

gB grain3Boundary
