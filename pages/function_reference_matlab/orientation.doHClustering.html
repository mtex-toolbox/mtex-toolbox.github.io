sort orientations into clusters

Syntax

{% highlight matlab %} [c,center] = doHCluster(ori,'numCluster',n) [c,center] = doHCluster(ori,'maxAngle',omega) {% endhighlight %}

Input

ori orientation
n number of clusters
omega maximum angle

Output

c list of clusters
center center of the clusters

Example

{% highlight matlab %} % generate orientation clustered around 5 centers cs = crystalSymmetry('m-3m'); center = orientation.rand(5,cs); odf = unimodalODF(center,'halfwidth',5*degree) ori = odf.discreteSample(3000); {% endhighlight %} {% highlight plaintext %} odf = SO3FunRBF (m-3m → xyz) multimodal components kernel: de la Vallee Poussin, halfwidth 5° center: 5 orientations Bunge Euler angles in degree phi1 Phi phi2 weight 294.445 7.14127 326.797 0.2 33.7384 31.2375 82.7667 0.2 71.8671 133.146 192.942 0.2 336.262 42.0429 288.184 0.2 240.198 77.0484 232.501 0.2 {% endhighlight %} {% highlight matlab %} % find the clusters and its centers tic; [c,centerRec] = calcCluster(ori,'method','hierarchical','numCluster',5); toc {% endhighlight %} {% highlight plaintext %} Elapsed time is 4.159629 seconds. {% endhighlight %} {% highlight matlab %} % visualize result oR = fundamentalRegion(cs) plot(oR) {% endhighlight %} {% highlight plaintext %} oR = orientationRegion crystal symmetry: 432 max angle: 62.7994° face normales: 14 vertices: 24 {% endhighlight %}
