arbitrary quaternion q with q * h = r

Description

The method fibre2quat defines a list of quaternion.quaternion.html,quaternions q by a crystal direction h and a specimen direction r such that q * h = r

Syntax

{% highlight matlab %} q = fibre2quat(h,r,varargin) {% endhighlight %}

Input

h Miller or vector3d
r vector3d

Output

q quaternion

Options

resolution double
