The class plane3d describes a pane in 3D space, given by the plane equation N.v = d

Syntax

{% highlight matlab %} plane = plane3d(N,d) plane = plane3d(N,v0) plane = plane3d.byPoints(v) {% endhighlight %}

Input

N normal direction vector3d
d distance to origin
v0, v vector3d

Output

plane plane3d
