--- title: isOuterBoundary sidebar: function_reference_sidebar permalink: grain2d.isOuterBoundary.html folder: function_reference toc: false --- isOuterBoundary edit page

check which of the indexed grains border with the map/ domain boundary - use this function to find boundary grains if they were recosntructed using alphaShapes

Syntax

{% highlight matlab %} out = isOuterBoundary(grains) {% endhighlight %}

Input

grains grain2d

Output

out logical

Options

'delta' double (default 0.95; 0 = convex hull,
1 = envelop touching all points)
see also isBoundary
