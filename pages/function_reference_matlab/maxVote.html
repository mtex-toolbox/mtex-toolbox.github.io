--- title: maxVote sidebar: function_reference_sidebar permalink: maxVote.html folder: function_reference toc: false --- maxVote edit page

returns

Syntax

{% highlight matlab %} maxInd = maxVote(ind,data) {% endhighlight %}

Input

idList list of ids
voteList list of votes

Output

vote most frequent vote in voteList(idList == id) for each id
numVotes number occurences of vote in voteList(idList == id) for each id

Options

strict only assign a vote if all votes coincide, otherwise set in to nan

Example

{% highlight matlab %} ind = [1,2,3,1,3,2,1]; data = [2,3,1,1,2,1,3]; maxInd = [7, 2, 5] maxInd = maxVote(ind,data) {% endhighlight %} {% highlight plaintext %} maxInd = 7 2 5 maxInd = 7 2 5 {% endhighlight %}
