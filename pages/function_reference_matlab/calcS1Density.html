compute periodic density on [0,2 pi]

Syntax

{% highlight matlab %} S1F = calcS1Density(x) {% endhighlight %}

Input

x double

Output

S1F S1FunHarmonic

Options

halfwidth width of the kernel function
antipodal ensures f(x+pi) = f(x)
even ensures f(x) = f(-x)
