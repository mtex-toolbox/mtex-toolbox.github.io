slice 3d-grains to obtain 2d-grains

Syntax

{% highlight matlab %} N = vector3d(1,1,1) % plane normal P0 = vector3d(0.5, 0.5, 0.5) % point within plane grain2d = grains.slice(N,P0) {% endhighlight %} {% highlight matlab %} plane = plane3d.fit(v) grain2d = grains.slice(plane) {% endhighlight %}

Input

grains3 grain3d
plane plane3d

Output

grains2 grain2d
