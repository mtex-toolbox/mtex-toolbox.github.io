give a regular grid in orientation space

Syntax

{% highlight matlab %} % sections according phi2 angle [S3G,S2G,phi2] = plotSO3Grid(CS,SS,'phi2') {% endhighlight %} {% highlight matlab %} % sections according rotational axis / angle [S3G,axes,omega,scaling] = plotSO3Grid(CS1,CS2,'axisAngle') {% endhighlight %}

Input

CS crystalSymmetry
SS specimenSymmetry

Output

S3G SO3Grid
S2G,axes list of vector3d, the plotting directions
phi2 double
scaling double
