--- title: grain2d.shapeFactor sidebar: function_reference_sidebar permalink: grain2d.shapeFactor.html folder: function_reference toc: false --- grain2d.shapeFactor edit page

calculates the shape factor of the grain-polygon, without Holes

define as

\[ F = \frac{P}{EP} \],

where \(P\) is the perimeter and \(EP\) is the equivalent perimeter.

Syntax

{% highlight matlab %} F = shapeFactor( grains ) {% endhighlight %}

Input

p grain2d

Output

F shape factor

See also

polygon.aspectRatio polygon.equivalentPerimeter polygon.perimeter
