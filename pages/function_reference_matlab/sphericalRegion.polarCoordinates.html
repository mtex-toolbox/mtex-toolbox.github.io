compute polar coordinates of with respect to a spherical region

Syntax

{% highlight matlab %} [r,rho] = polarCoordinates(sR,v,center,ref,varargin) {% endhighlight %}

Input

sR sphericalRegion
v vector3d
center vector3d

Output

r relative distance to the center
rho angle with respect to the center
