compute the average tensor for an EBSD data set

Syntax

{% highlight matlab %} % applies a tensor on a given phase [TVoigt, TReuss, THill] = calcTensor(ebsd('phase2'),T_phase2) {% endhighlight %} {% highlight matlab %} % applies each tensor given in order of input to each phase [TVoigt, TReuss, THill] = calcTensor(ebsd,T_phase1,T_phase2,...) {% endhighlight %} {% highlight matlab %} % returns the specified tensor THill = calcTensor(ebsd,T_phase1,T_phase2,'Hill') {% endhighlight %} {% highlight matlab %} % geometric mean instead of arithmetic one TGeom = calcTensor(ebsd,T_phase1,T_phase2,'geometric') {% endhighlight %}

Input

ebsd EBSD
T_phaseN tensor for the N-th phase

Output

T tensor

Options

Voigt Voigt mean
Reuss Reuss mean
Hill Hill mean
geometric geometric mean
