disjoint list of quaternions

Syntax

{% highlight matlab %} u = unique(q) u = unique(q,'tolerance',tol) [u,iq,iu] = unique(q) {% endhighlight %}

Input

q quaternion
tol double (default 1e-3)

Output

u quaternion
iq index such that u = q(iq)
iu index such that q = u(iu)

Flags

stable prevent sorting
