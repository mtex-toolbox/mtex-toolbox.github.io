define orientations by Miller Bravais indeces

Description

Defines an orientation ori by Miller indece (hkl) and [uvw] such that ori * (hkl) = 001 and ori * [uvw] = 100

Syntax

{% highlight matlab %} hkl = Miller(h,k,l,cs,'hkl') uvw = Miller(u,v,w,cs,'uvw') ori = orientation.byMiller(hkl,uvw) {% endhighlight %} {% highlight matlab %} ori = orientation.byMiller([h k l],[u v w],cs) {% endhighlight %}

Input

hkl, uvw Miller
h,k,l Miller indece (double)
u,v,w Miller indece (double)
cs crystalSymmetry

Output

ori orientation
