plots odf or append to a previous plot using 'add2all'

Syntax

{% highlight matlab %} % plot in phi2 sections plot(odf) {% endhighlight %} {% highlight matlab %} % plot in specific phi2 sections plot(odf,'phi2',45*degree) {% endhighlight %} {% highlight matlab %} % plot in 3d space plot(odf,'axisAngle') plot(odf,'rodrigues') {% endhighlight %} {% highlight matlab %} % plot along a fibre f = fibre.alpha(odf.CS) plot(odf,f) {% endhighlight %} {% highlight matlab %} % plot the odf as sigma sections oS = sigmaSections(odf.CS) plot(odf,oS) {% endhighlight %}
