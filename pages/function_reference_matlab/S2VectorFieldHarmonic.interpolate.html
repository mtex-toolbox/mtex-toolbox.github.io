Interpolate an S2VectorField by given tangent vectors at given points on the sphere.

Syntax

{% highlight matlab %} sVF = S2VectorField.interpolate(v, value) sVF = S2VectorField.interpolate(v, value, 'bandwidth', bw) {% endhighlight %}

Input

value vector3d
v vector3d

Output

sVF S2VectorFieldHarmonic

Options

bw degree of the spherical harmonic (default: 128)
