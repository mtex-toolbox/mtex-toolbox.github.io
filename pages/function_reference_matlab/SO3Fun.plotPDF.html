plot pole density function

Syntax

{% highlight matlab %} plotPDF(odf,[h1,..,hN]) plotPDF(odf,{[h11,h12],h2,hN],'superposition',{[c11,c12],c2,cN}) plotPDF(odf,pf.h,'superposition',pf.c) {% endhighlight %}

Input

odf SO3Fun
h Miller crystallographic directions
c structure coefficients

Options

resolution resolution of the plots
superposition plot superposed pole figures

Flags

noTitle suppress the Miller indices at the top
antipodal include antipodal symmetry
complete plot entire (hemi)--sphere
