define rotentations by Euler angles

Syntax

{% highlight matlab %} rot = rotation.byEuler(phi1,Phi,phi2) % Bunge convention rot = rotation.byEuler(alpha,beta,gamma,'ZYZ') % Matthies convention rot = rotation.byEuler(phi1,Phi,phi2,'Kocks') % Kocks convention {% endhighlight %}

Input

phi1, Phi, phi2 Euler angles in radiant

Output

rot rotation

Flags

Bunge, ZXZ
ABG, Matthies, ZYZ
Roe
Kocks
Canova
