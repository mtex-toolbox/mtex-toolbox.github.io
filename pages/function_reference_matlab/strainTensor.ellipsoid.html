--- title: ellipsoid sidebar: function_reference_sidebar permalink: strainTensor.ellipsoid.html folder: function_reference toc: false --- ellipsoid edit page

Length (1+e) and direction of principal axes of strain ellipsoid from finite, lagrangian/eulerian strain tensor E with eigenvalues lambda with (1+e) = sqrt(1+2*lambda) for a Lagranage strain tensor (default) (1+e) = sqrt(1-2*lambda) for a Euler strain tensor

Syntax

{% highlight matlab %} [fe,qe] = ellipsoid(E) {% endhighlight %}

Input

E strainTensor

Output

fe directions of prinicpal axes of strain ellipsoid vector3d
qe length of ellipse axes (1+e)
