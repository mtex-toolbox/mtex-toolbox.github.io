rotate EBSD maps

Syntax

{% highlight matlab %} % rotate EBSD map about the z-axis by 10*degree ebsd = rotate(ebsd,10*degree) {% endhighlight %} {% highlight matlab %} % rotate about the x-axis ebsd = rotate(ebsd,rotation.byAxisAngle(xvector,180*degree)) {% endhighlight %} {% highlight matlab %} % rotate only the spatial data ebsd = rotate(ebsd,180*degree,'keepEuler') {% endhighlight %} {% highlight matlab %} % rotate about a specific point ebsd = rotate(ebsd,180*degree,'center',[0,0]) {% endhighlight %}

Input

ebsd EBSD
angle double
q rotation

Output

ebsd EBSD

Options

center [x,y] center of rotation, default is (0,0)

Flags

keepXY rotate only the orientation data, i.e. the Euler angles
keepEuler rotate only the spatial data, i.e., the x,y, and z values
