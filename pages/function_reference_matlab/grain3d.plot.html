--- title: grain3d.plot sidebar: function_reference_sidebar permalink: grain3d.plot.html folder: function_reference toc: false --- grain3d.plot edit page

plot 3-dimensional grains, or more precise their faces

Only outer faces, the inner grains have the color gray [0.5 0.5 0.5].

Syntax

{% highlight matlab %} plot(grains) % colorize by phase plot(grains,property) % colorize by property plot(grains,property,'LineStyle','none') % adapt LineStyle {% endhighlight %}

Input

grains grain3d
property n x 1 numeric or orientation
