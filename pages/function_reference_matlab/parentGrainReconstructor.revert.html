revert parent orientations to measured child orientations

Syntax

{% highlight matlab %} % undo all reconstruction job.revert {% endhighlight %} {% highlight matlab %} % undo grains by condition ind = job.grains.fit > 5*degree job.revert(ind) {% endhighlight %} {% highlight matlab %} % undo specific grains job.revert(job.parentGrains(5)) {% endhighlight %}

Input

job parentGrainReconstructor
ind true/false of which parent grains should be reverted to child grains
grains list of grains to be reverted

Output

job parentGrainReconstructor
