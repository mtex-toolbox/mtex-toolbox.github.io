Approximate a vector field on the rotation group (SO(3)) in its multimodal RBF-Kernel representation from a given vector field.

We compute this vector field componentwise, i.e. we compute three SO3FunRBFs individually by approximation.

Syntax

{% highlight matlab %} SO3VF = SO3VectorFieldRBF.approximate(f) {% endhighlight %}

Input

f SO3VectorField

Output

SO3VF SO3VectorFieldRBF
