compute the angle distribution of a uniform ODF for a crystal symmetry

Syntax

{% highlight matlab %} [ad,omega] = calcAngleDistribution(cs1) [ad,omega] = calcAngleDistribution(cs1,cs2) [ad,omega] = calcAngleDistribution(cs1,omega) {% endhighlight %}

Input

cs crystalSymmetry
omega angle

Output

ad angle distribution
omega angles

Options

angle|threshold distribution with the angles within a threshold
