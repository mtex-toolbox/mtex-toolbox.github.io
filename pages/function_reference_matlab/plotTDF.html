Wrapper for the Matlab polar plot, can be used with calcTDF

Syntax

{% highlight matlab %} p1 = plotTDF(bc,pd,varargin) {% endhighlight %}

Input

bc azimuth value
pd polar value

Options

nogrid get rid of grid on polar plot
nolabels get rid of polar labels
linewidth
linecolor
linestyle
