--- title: vector3d.nanmean sidebar: function_reference_sidebar permalink: vector3d.nanmean.html folder: function_reference toc: false --- vector3d.nanmean edit page

computes the mean vector, sets nan vectors to 0

Syntax

{% highlight matlab %} % average direction with respect to the first nonsingleton dimension m = mean(v) {% endhighlight %} {% highlight matlab %} % average direction along dimension d m = mean(v,d) {% endhighlight %} {% highlight matlab %} % average axis m = mean(v,'antipodal') {% endhighlight %}

Input

v vector3d

Output

m vector3d

Options

antipodal include antipodal symmetry
robust robust mean (with respect to outliers)
