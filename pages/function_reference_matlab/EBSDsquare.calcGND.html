compute the geometrically necessary dislocation

Formulae are taken from the paper:

Pantleon, Resolving the geometrically necessary dislocation content by conventional electron backscattering diffraction, Scripta Materialia, 2008

Syntax

{% highlight matlab %} gnd = calcGND(ebsd,dS) {% endhighlight %}

Input

ebsd @EBSDSquare
dS dislocationSystem

Output

gnd dislocation energy
rho dislocation density per dislocation system
