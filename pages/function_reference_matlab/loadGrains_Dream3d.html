loadGrains_Dream3d is a method to load 3d grain data from dream3d

Syntax

{% highlight matlab %} grains = grain3d.load('filepath/filename.dream3d','interface','dream3d) grains = loadGrains_Dream3d('filepath/filename.dream3d') {% endhighlight %}

Input

fname filename

Output

grain3d grain3d
