smooth spatial EBSD

Syntax

{% highlight matlab %} ebsd = smooth(ebsd) {% endhighlight %} {% highlight matlab %} F = halfQuadraticFilter F.alpha = 2; ebsd = smooth(ebsd, F, 'fill', grains) {% endhighlight %}

Input

ebsd EBSD
F EBSDFilter
grains grain2d if provided pixels at the boundary between grains are not filled

Options

fill fill missing values (this is different then not indexed values!)
extrapolate extrapolate up the the outer boundaries

Example

{% highlight matlab %} mtexdata forsterite; ebsd = ebsd('indexed'); % segment grains [grains,ebsd.grainId] = calcGrains(ebsd,'minPixel',4); {% endhighlight %} {% highlight plaintext %} ebsd = EBSD (y↑→x) Phase Orientations Mineral Color Symmetry Crystal reference frame 0 58485 (24%) notIndexed 1 152345 (62%) Forsterite LightSkyBlue mmm 2 26058 (11%) Enstatite DarkSeaGreen mmm 3 9064 (3.7%) Diopside Goldenrod 12/m1 X||a*, Y||b*, Z||c Properties: bands, bc, bs, error, mad Scan unit : um X x Y x Z : [0, 36550] x [0, 16750] x [0, 0] Normal vector: (0,0,1) {% endhighlight %}

find largest grains

{% highlight matlab %} largeGrains = grains(grains.numPixel>800); ebsd = ebsd(largeGrains(1)); {% endhighlight %} {% highlight matlab %} figure plot(largeGrains(1).boundary,'linewidth',2) hold on oM = ipfHSVKey(ebsd); oM.inversePoleFigureDirection = mean(ebsd.orientations) * oM.whiteCenter; oM.colorStretching = 50; plot(ebsd,oM.orientation2color(ebsd.orientations)) hold off {% endhighlight %}
