a class representing radial ODFs by an SO3Kernel function and center orientations

Syntax

{% highlight matlab %} SO3F = SO3FunRBF(center,psi,weights,c0) SO3F = SO3FunRBF(F) SO3F = SO3FunRBF(F,'kernel',psi,__) % See SO3FunRBF.approximate for options list SO3F = SO3FunRBF(F,'harmonic',__) % See SO3FunRBF.approximate for option list {% endhighlight %}

Input

center orientation
psi SO3Kernel
weights double
c0 double
F SO3Fun

Output

SO3F SO3FunRBF
