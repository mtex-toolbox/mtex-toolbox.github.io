regular grid in Euler orientation space

Syntax

{% highlight matlab %} SO3G = regularSO3Grid(cs) SO3G = regularSO3Grid(cs,ss,'resolution',2.5*degree) % specify the resolution SO3G = regularSO3Grid(cs,ss,'resolution',5*degree,'ZYZ') % use ZYZ convention SO3G = regularSO3Grid(cs,ss,'phi2','sections',10) % 10 phi2 sections SO3G = regularSO3Grid('ClenshawCurtis','bandwidth',64) % {% endhighlight %}

Input

cs crystalSymmetry
ss specimenSymmetry

Output

SO3G orientation

Options

resolution Euler angle resolution
sections number of sections

Flags

phi1 | Phi | phi2 sections along which variable
