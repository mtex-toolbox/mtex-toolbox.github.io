rotational axis of an misorientation or two orientations

Syntax

{% highlight matlab %} % define a cubic and hexagonal symmetries csCube = crystalSymmetry('cubic') csHex = crystalSymmetry('hexagonal') {% endhighlight %} {% highlight matlab %} % define two orientations o1 = orientation.byEuler(0,0,0,csCube) o2 = orientation.byEuler(10*degree,20*degree,30*degree,csHex) {% endhighlight %} {% highlight matlab %} % the misorientation axis with respect to the specimen coordinate % system is computed by a = axis(o1,o2) {% endhighlight %} {% highlight matlab %} % the misorientation axis with respect to csCube is computed by a = axis(inv(o1)*o2,csCube) {% endhighlight %} {% highlight matlab %} % the misorientation axis with respect to csHex is computed by a = axis(inv(o1)*o2,csHex) {% endhighlight %} {% highlight matlab %} % compute the misorientation axis ignoring symmetry a = axis(inv(o1)*o2,'noSymmetry') {% endhighlight %}

Input

mori,o1,o2 orientation

Output

a vector3d
m Miller

Options

noSymmetry ignore symmetry
max return the axis corresponding to the largest rotational angle
