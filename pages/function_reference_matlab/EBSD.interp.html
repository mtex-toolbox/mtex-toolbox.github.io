interpolate EBSD map at arbitrary points positions

Syntax

{% highlight matlab %} ebsdNew = interp(ebsd,pos) {% endhighlight %} {% highlight matlab %} ebsdNew = interp(ebsd,xNew,yNew,'method','invDist') {% endhighlight %}

Input

ebsd EBSD
pos vector3d
xNew, yNew new x,y coordinates

Output

ebsdNew EBSD with coordinates (xNew,yNew)

Options

method 'invDist', 'nearest'
