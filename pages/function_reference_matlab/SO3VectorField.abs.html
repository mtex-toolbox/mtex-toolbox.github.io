overloads the componentwise absolute value abs(SO3VF)

Note that abs is not the length of a vector3d which is the evaluation of the SO3VectorField in one rotation.

one

Syntax

{% highlight matlab %} SO3VF = abs(SO3VF) {% endhighlight %}

Input

SO3VF SO3VectorField

Output

SO3VF SO3VectorField
