grain3d.load is a method to load 3d grain data

Syntax

{% highlight matlab %} grains = grain3d.load('filepath/filename.tess','CS',CSList) grains = grain3d.load('filepath/filename.dream3d') {% endhighlight %}

Input

fname filename
CSList list of crystal symmetries

Output

grain3d grain3d
