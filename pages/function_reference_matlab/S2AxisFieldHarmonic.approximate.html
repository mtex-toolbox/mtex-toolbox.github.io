computes an approximation from a given spherical axis field by computing the spherical Fourier coefficients with quadrature componentwise.

Syntax

{% highlight matlab %} sAF = S2AxisFieldHarmonic.approximate(f) sAF = S2AxisFieldHarmonic.approximate(f, 'bandwidth', bw) {% endhighlight %}

Input

f function handle in vector3d

Output

sAF S2AxisFieldHarmonic

Options

bw degree of the spherical harmonic (default: 128)
