disjoint list of rotations

Syntax

{% highlight matlab %} u = unique(r) u = unique(r,'tolerance',0.01) [u,ir,iu] = unique(r) {% endhighlight %}

Input

r rotation
tol double (default 1e-3)

Output

u rotation
ir index such that u = r(ir)
iu index such that r = u(iu)

Flags

stable prevent sorting
