DISPLAY Display array. DISPLAY(X) is called for the object X when the semicolon is not used to terminate a statement.

{% highlight matlab %} For example, X = datetime(2014,1:5,1) calls DISPLAY(X) while X = datetime(2014,1:5,1); does not. {% endhighlight %} {% highlight matlab %} A typical implementation of DISPLAY calls DISP to do most of the work. To customize the display of objects, overload the DISP function. Note that DISP does not display empty arrays. {% endhighlight %}

Syntax

{% highlight matlab %} display(job,varargin) {% endhighlight %}
