the logarithmic map that translates a rotation into a spin tensor

Syntax

{% highlight matlab %} Omega = logm(mori) % spin tensor of a misorientation {% endhighlight %} {% highlight matlab %} Omega = logm(ori,ori_ref) % spin tensor in crystal coordinates Omega = logm(ori,ori_ref,'left') % spin tensor in specimen coordinates {% endhighlight %}

Input

mori misorientation
ori orientation
ori_ref orientation

Output

Omega spinTensor
