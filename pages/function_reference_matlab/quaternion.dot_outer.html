outer inner product between two quaternions

Description

cos angle(g1,g2)/2 = dot(g1,g2)

Syntax

{% highlight matlab %} d = dot_outer(g1,g2,varargin) {% endhighlight %}

Input

g1, g2 quaternion

Output

d double
