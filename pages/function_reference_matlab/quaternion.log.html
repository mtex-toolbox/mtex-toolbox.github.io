the logarithmic map that translates a rotation into a rotation vector

Syntax

{% highlight matlab %} v = log(q) % rotation vector with reference to the identical rotation v = log(q,q_ref) % rotation vector with reference q_ref {% endhighlight %} {% highlight matlab %} v = loq(q,'right') v = loq(q,'left') {% endhighlight %}

Input

q quaternion
q_ref quaternion

Output

v vector3d
