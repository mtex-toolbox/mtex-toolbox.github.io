grain3d.intersected is a method to determine if a 3D grain data cell is intersected by a plane in 3D space

Syntax

{% highlight matlab %} N = vector3d(1,1,1) % plane normal P0 = vector3d(0.5, 0.5, 0.5) % point within plane is_intersected = grains.intersected(N,P0) {% endhighlight %} {% highlight matlab %} V = vector3d([0 1 0],[0 1 1],[0 1 0]) is_intersected = grains.intersected(V) % set of points grains = grains(is_intersected) {% endhighlight %}

Input

grains grain3d
plane plane3d

Output

out logical
