evaluate sF on ori via moving least squares (MLS) approximation provide the possibility of also returning the condition numbers of the gram matrices

Syntax

{% highlight matlab %} vals = SO3F.eval(ori) vals = eval(SO3F,ori) {% endhighlight %}

Input

SO3F the function we want to approximate
ori the points where we want to evaluate the MLS approximation

Output

vals the values of SO3F on ori
