eigenvalue and vectors of a symmetric 3x3 matrix

Syntax

{% highlight matlab %} lambda = eig3(M) {% endhighlight %} {% highlight matlab %} lambda = eig3(a11,a12,a13,a22,a23,a33) {% endhighlight %} {% highlight matlab %} [v,lambda] = eig3(a11,a12,a13,a22,a23,a33) {% endhighlight %}

Input

M array of symmetric 3x3 matrix
a11, a12,a13,a22,a23,a33 vector of matrix elements

Output

lambda eigen values
v eigen vectors
