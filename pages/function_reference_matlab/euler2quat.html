converts euler angle to quaternion

Description

The method euler2quat defines a quaternion by Euler angles. You can choose whether to use the Bunge (phi,Phi,phi2) convention or the Matthies (alpha,beta,gamma) convention.

Syntax

{% highlight matlab %} q = euler2quat(alpha,beta,gamma) - q = euler2quat(phi1,Phi,phi2,'Bunge') - {% endhighlight %}

Input

alpha, beta, gamma double
phi1, Phi, phi2 double

Output

q quaternion

Options

ABG, ZYZ Matthies (alpha, beta, gamma) convention (default)
BUNGE, ZXZ Bunge (phi1,Phi,phi2) convention
