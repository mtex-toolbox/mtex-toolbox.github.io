convert x,y,z coordinates into indices of EBSD3square

Syntax

{% highlight matlab %} ind = pos2ind(ebsd,pos) ebsd(ind) {% endhighlight %} {% highlight matlab %} [i,j,k] = pos2ind(ebsd,x,y,z) ebsd(i,j,k) {% endhighlight %}

Input

ebsd EBSD3square
x,y,z spatial coordinates
pos vector3d

Output

ind index to EBSD3square
i,j,k indices to EBSD3square
