rotate grainBoundaries

Syntax

{% highlight matlab %} % rotate the gB about the z-axis by 90*degree gB = rotate(gB,10*degree) {% endhighlight %} {% highlight matlab %} % rotate about the x-axis gB = rotate(gB,rotation.byAxisAngle(xvector,180*degree)) {% endhighlight %} {% highlight matlab %} % rotate about a specific point gB = rotate(gB,180*degree,'center',[0,0]) {% endhighlight %}

Input

gB grainBoundary
angle double
rot rotation

Output

gB grainBoundary

Options

center [x,y] center of rotation, default is center of the gB

Flags

keepXY rotate only the orientation data, i.e. the Euler angles
keepEuler rotate only the spatial data, i.e., the x,y, and z values
