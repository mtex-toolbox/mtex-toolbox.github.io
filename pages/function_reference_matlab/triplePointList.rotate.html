rotate triple points

Syntax

{% highlight matlab %} % rotate the tP about the z-axis by 90*degree tP = rotate(tP,10*degree) {% endhighlight %} {% highlight matlab %} % rotate about the x-axis tP = rotate(tP,rotation.byAxisAngle(xvector,180*degree)) {% endhighlight %} {% highlight matlab %} % rotate about a specific point tP = rotate(tP,180*degree,'center',[0,0]) {% endhighlight %}

Input

tP triplePointList
angle double
rot rotation

Output

tP triplePointList

Options

center [x,y] center of rotation, default is (0,0)

Flags

keepXY rotate only the orientation data, i.e. the Euler angles
keepEuler rotate only the spatial data, i.e., the x,y, and z values
