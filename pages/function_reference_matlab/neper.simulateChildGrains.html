simulate a 3d microstructure after martensitic phase transformation

Syntax

{% highlight matlab %} neper.init numGrains = 100; odf = unimodalODF(orientation.rand); grains = neper.simulateChildGrains(numGrains, p2c, odfParent, habitPlane) {% endhighlight %}

Input

numGrains number of grains
p2c parent to child OR
odfParent SO3Fun
habitPlane Miller
'silent' print log file, no console output

Output

grains grain3d
