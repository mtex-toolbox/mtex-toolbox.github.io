define a unimodal ODF

Description

unimodalODF defines a radially symmetric, unimodal ODF with respect to a crystal orientation mod. The shape of the ODF is defined by a SO3Kernel function.

Syntax

{% highlight matlab %} mod = orientation.byEuler(phi1,Phi,phi2,CS,SS) odf = unimodalODF(mod) % default halfwidth 10 degree odf = unimodalODF(mod,'halfwidth',15*degree) % specify halfwidth odf = unimodalODF(mod,CS,SS) % specify crystal and specimen symmetry odf = unimodalODF(mod,psi) % specify shape odf = unimodalODF(mod,'weights',weights) % specify weights for each component {% endhighlight %}

Input

mod quaternion modal orientation
CS, SS crystal, specimen symmetry
hw halfwidth of the kernel (default -- 10Â°)
psi SO3Kernel function (default -- SO3 de la Vallee Poussin)

Output

odf SO3Fun

Flags

nothinning keep all nodes, otherwise small portions will be thinned out by default
