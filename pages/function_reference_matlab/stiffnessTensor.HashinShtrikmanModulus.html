Hashin Shtrikman moduli

Description

The equations are from Peselnick and Meister 1965 through Watt and Peselnick 1980 which are based on Hashin and Shtrikman 1963 JMB 8/2013

Syntax

{% highlight matlab %} [khs, ghs, def] = HashinShtrikmanModulus(C,Ko,Go) {% endhighlight %}

Input

C stiffnessTensor
Ko bulk modulus of the reference isotropic material
Go shear modulus of the reference isotropic material

Output

khs Hashin Shtrikman bulk modulus
ghs Hashin Shtrikman shear modulus
def +1 positive definite, -1 for negative definite R
