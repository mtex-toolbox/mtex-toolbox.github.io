the logarithmic map that translates a rotation into a spin tensor

Syntax

{% highlight matlab %} T = logm(q) % spin tensor with reference to the identical rotation T = logm(q,q_ref) % spin tensor with reference q_ref {% endhighlight %}

Input

q quaternion
q_ref quaternion

Output

T spinTensor
