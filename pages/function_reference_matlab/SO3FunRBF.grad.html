left-sided gradient of an SO3Fun

Description

general formula:

\[s(g1_i) = sum_j c_j DK(g1_i,g2_j) \]

Syntax

{% highlight matlab %} G = SO3F.grad % compute the gradient g = SO3F.grad(rot) % evaluate the gradient in rot {% endhighlight %} {% highlight matlab %} % go 5 degree in direction of the gradient ori_new = exp(rot,5*degree*normalize(g)) {% endhighlight %}

Input

SO3F SO3FunRBF
rot rotation / orientation

Output

G SO3VectorField
g vector3d
