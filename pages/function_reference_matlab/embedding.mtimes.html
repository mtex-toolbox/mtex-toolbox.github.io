implements e * c and c * e

Description

If e is a matrix of embeddings and c is a matrix of coefficients then e * c is again a matrix of embeddings defined by

\[ [\mathrm{e * c}]_{j\ell} = \sum_{k} \mathrm{e}_{jk} \mathrm{c}_{k \ell}\]

Syntax

{% highlight matlab %} out = e * c out = c * e {% endhighlight %}

Input

e embedding
c double

Output

out embedding
