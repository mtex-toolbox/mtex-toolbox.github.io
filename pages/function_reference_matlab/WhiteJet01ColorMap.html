WHITEJET01COLORMAP fix bottom of jet colourmap at 1 and white at 0 when plotting ODFs with WhiteJetColorMap it might be useful to define an additional fixed point at MUD = 1 to compare texture strengths 1 (uniform ODF / random texture) = dark blue (min in Jet colormap) cmax (ODF mode / texture peak) = red (max in Jet colormap) 0 (never found in ODF) = white

Syntax

{% highlight matlab %} [map] = WhiteJet01ColorMap(cmax,varargin) {% endhighlight %}
