find ind such that ebsd.id(ind) == id

Syntax

{% highlight matlab %} ind = id2ind(ebsd,id) {% endhighlight %}

Input

ebsd EBSD
id a list of id's as stored in ebsd.id

Output

ind a list indices such that ebsd.id(ind) == id
