convert x,y,z coordinates into indices of ebsd

Syntax

{% highlight matlab %} ind = pos2ind(ebsd,x,y) ebsd(ind) {% endhighlight %} {% highlight matlab %} [i,j] = pos2ind(ebsd,x,y,z) ebsd(i,j) {% endhighlight %} {% highlight matlab %} ind = pos2ind(ebsd,pos) {% endhighlight %}

Input

ebsd EBSDsquare
x,y,z spatial coordinates
pos vector3d

Output

ind index to EBSDsquare
i,j indices to EBSDsquare
