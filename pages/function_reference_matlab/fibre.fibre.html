--- title: fibre.fibre sidebar: function_reference_sidebar permalink: fibre.fibre.html folder: function_reference toc: false --- fibre.fibre edit page

Class representing a fibre in orientation space. Examples are alpha, beta or gamma fibres. In general a fibre is defined by a crystal direction h of type Miller and a specimen direction of type vector3d.

Syntax

{% highlight matlab %} cs = crystalSymmetry('432') f = fibre.alpha(cs,'full') % the alpha fibre {% endhighlight %} {% highlight matlab %} plotPDF(f,Miller(1,0,0,cs)) {% endhighlight %} {% highlight matlab %} f = fibre(o1,o2) % the fibre from o1 to o2 {% endhighlight %} {% highlight matlab %} f = fibre(Miller(0,0,1,cs),vector3d.Z,r) % the fibre (001)||ND f = fibre(h,r) % the fibre with h parallel to r {% endhighlight %}

Input

cs crystalSymmetry
o1, o2 orientation
h Miller
r vector3d

Class Properties

o1, o2 start and end orientation
h Miller
