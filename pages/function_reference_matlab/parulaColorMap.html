PARULA Blue-green-orange-yellow color map PARULA(M) returns an M-by-3 matrix containing a colormap. The colors begin with dark purplish-blue and blue, range through green and orange, and end with bright yellow. PARULA is named after a bird, the tropical parula, which has these colors.

{% highlight matlab %} PARULA returns a colormap with the same number of colors as the current figure's colormap. If no figure exists, MATLAB creates one. {% endhighlight %} {% highlight matlab %} EXAMPLE {% endhighlight %} {% highlight matlab %} This example shows how to reset the colormap of the current figure. {% endhighlight %} 
     colormap(parula)

Syntax

{% highlight matlab %} map = parula(n) {% endhighlight %}
