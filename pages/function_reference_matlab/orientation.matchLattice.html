find lattice alignments for arbitrary misorientations

Description

Given a misorientation mori find all face normals n1, n2 and crystal directions d1, d2, such that mori * n1 = n2 and mori * d1 = d2.

Syntax

{% highlight matlab %} matchLattice(mori) {% endhighlight %} {% highlight matlab %} [n1,n2,d1,d2] = matchLattice(mori) [n1,n2,d1,d2] = matchLattice(mori,'threshold',1*degree) [n1,n2,d1,d2] = matchLattice(mori,'maxIndex',3) {% endhighlight %}

Input

mori misorientation

Output

uvw,hkl Miller
n1,n2,d1,d2 Miller

Example

revert sigma3 misorientation relationship

{% highlight matlab %} [n1,n2,d1,d2] = matchLattice(CSL(3,crystalSymmetry('432'))) {% endhighlight %} {% highlight plaintext %} n1 = Miller (432) size: 37 x 1 n2 = Miller (432) size: 37 x 1 d1 = Miller (432) size: 37 x 1 d2 = Miller (432) size: 37 x 1 {% endhighlight %}
