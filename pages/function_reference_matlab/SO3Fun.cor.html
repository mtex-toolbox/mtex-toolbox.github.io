correlation between two SO3Funs

\[ cor(f_1,f_2) = \sqrt{\frac1{8\pi^2} \int_{SO(3)} f_1® f_2® dR\],

Syntax

{% highlight matlab %} c = cor(SO3F1,SO3F2) {% endhighlight %}

Input

SO3F1, SO3F2 SO3Fun

Output

c double
