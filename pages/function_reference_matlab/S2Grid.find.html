return index of all points in a epsilon neighborhood of a vector

Syntax

{% highlight matlab %} ind = find(S2G,v,epsilon) % find all points in a epsilon neighborhood of v ind = find(S2G,v) % find closest point {% endhighlight %}

Input

S2G S2Grid
v vector3d
epsilon double

Output

ind int32

Options

antipodal include antipodal symmetry
