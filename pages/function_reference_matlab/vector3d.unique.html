disjoint list of vectors

Syntax

{% highlight matlab %} u = unique(v) % find disjoint elements of the vector v u = unique(v,'tolerance',0.01) % use tolerance 0.01 [u,iv,iu] = unique(v,varargin)] {% endhighlight %}

Input

v vector3d
tol double (default 1e-7)

Output

u vector3d
ir index such that u = r(ir)
iv index such that r = u(iv)

Flags

stable prevent sorting
antipodal tread vectors as axes
noAntipodal ignore antipodal symmetry
