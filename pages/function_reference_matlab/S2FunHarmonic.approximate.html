computes an approximation from a given spherical function by computing the spherical Fourier coefficients with quadrature

Syntax

{% highlight matlab %} sF = S2FunHarmonic.approximate(f) sF = S2FunHarmonic.approximate(f, 'bandwidth', bandwidth) {% endhighlight %}

Input

f function on the sphere (may be multidimensional)

Options

bandwidth maximum degree of the spherical harmonics used to approximate the function
