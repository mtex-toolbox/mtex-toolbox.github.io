Compute the adjoint S2-Fourier transform of given evaluations on a specific quadrature grid, by using the NFSFT-method (nonequispaced fast spherical fourier transform).

Syntax

{% highlight matlab %} sF = S2FunHarmonic.adjoint(vec,values) sF = S2FunHarmonic.adjoint(vec,values,'bandwidth',32,'weights',w) sF = S2FunHarmonic.adjoint(f) {% endhighlight %}

Input

vec vector3d
values double

Output

sF S2FunHarmonic

Options

bandwidth maximal harmonic degree (default: 128)
weights quadrature weights
