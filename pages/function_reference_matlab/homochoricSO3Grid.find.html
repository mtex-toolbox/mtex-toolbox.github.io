return indece and distance of all nodes within a eps neighborhood

Syntax

{% highlight matlab %} % find the closes point [ind,dist] = find(SO3G,ori) {% endhighlight %} {% highlight matlab %} % find points with a radius [ind,dist] = find(SO3G,ori,radius) {% endhighlight %} {% highlight matlab %} % find cube corners cubeInd = find(SO3G,ori,'cube') {% endhighlight %}

Input

SO3G homochoricSO3Grid
ori orientation
radius double

Output

ind index of the closes grid point
cubeInd Nx8 list of indeces of the cube corners containing ori
dist misorientation angle
