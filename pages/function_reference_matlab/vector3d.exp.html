rotation vector to rotation

Syntax

{% highlight matlab %} mori = exp(v) % misorientation in specimen coordinates {% endhighlight %} {% highlight matlab %} rot = exp(v,ori_ref,SO3TangentSpace.rightVector) % orientation update {% endhighlight %} {% highlight matlab %} ori = exp(v,ori_ref,tS) % orientation update {% endhighlight %}

Input

v vector3d, SO3TangentVector
ori_ref orientation, rotation
tS SO3TangentSpace

Output

mori rotation
ori orientation
