symmetrises a function with respect to a symmetry or a direction

Syntax

{% highlight matlab %} % symmetrise with respect to a crystal or specimen symmetry sFs = symmetrise(sF,cs) sFs = symmetrise(sF,ss) {% endhighlight %} {% highlight matlab %} % symmetrise with respect to an axis [sFs,psi] = symmetrise(sF,d) {% endhighlight %}

Input

sF S2FunHarmonic
cs,ss crystalSymmetry, specimenSymmetry
d vector3d

Output

sFs S2FunHarmonic
psi S2Kernel
