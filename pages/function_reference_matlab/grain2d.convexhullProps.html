Area of the convex hull of grains Note: this is only reasonable for smooth and sufficiently large grains

Syntax

{% highlight matlab %} [cha, chp] = convexhullArea(grains) {% endhighlight %}

Input

grains grain2d

Output

cha area of convex hull
chp perimeter of convex hull
