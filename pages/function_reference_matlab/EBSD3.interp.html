Syntax

{% highlight matlab %} ebsdNew = interp(ebsd,newPos) {% endhighlight %} {% highlight matlab %} ebsdNew = interp(ebsd,newPos,'method','invDist') {% endhighlight %}

Input

ebsd EBSD3

Output

ebsdNew EBSD with coordinates (xNew,yNew)

Options

method 'invDist', 'nearest'
