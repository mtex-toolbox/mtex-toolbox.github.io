plot a grain size histogram

Syntax

{% highlight matlab %} hist(grains) hist(grains,n) % specify the number of bins {% endhighlight %} {% highlight matlab %} % use an arbitrary property for the histogram hist(grains,grains.equivalentRadius) {% endhighlight %} {% highlight matlab %} binId = hist(grains) % returns the bin for each grain {% endhighlight %}

Input

grains grain2d
n number of bins, default is 15
