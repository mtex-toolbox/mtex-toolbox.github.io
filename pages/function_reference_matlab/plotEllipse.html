plot multiple ellipses

Syntax

{% highlight matlab %} [a,b] = principalComponents(grains); plotEllipse(grains.centroid,a,b,'lineColor','r') {% endhighlight %}

Input

c center of the ellipse vector3d
a,b half axes of the ellipse vector3d

Options

lineColor colorspec
