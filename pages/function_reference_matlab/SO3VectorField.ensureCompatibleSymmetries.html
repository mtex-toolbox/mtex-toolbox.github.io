For calculating with @SO3VectorFields (+, -, .*, ./, ...) we have to verify that the symmetries are suitable.

Syntax

{% highlight matlab %} ensureCompatibleSymmetries(SO3F1,SO3F2) ensureCompatibleSymmetries(SO3F1,sF) ensureCompatibleSymmetries(SO3F1,ori) ensureCompatibleSymmetries(SO3F1,SO3F2,'conv') ensureCompatibleSymmetries(SO3F1,'antipodal') {% endhighlight %}

Input

SO3F1, SO3F2 SO3VectorField, SO3Fun
sF S2FunHarmonicSym
ori orientation

Output

msg yields a error message if the symmetry do not match

Options

conv be shure switched symmetries match
