constructor

tensor is the low level constructor for a tensor object. For importing real world data you might want to use the import_wizard.

Syntax

{% highlight matlab %} T = tensor(M,CS,'name',name,'unit',unit,'propertyname',property) {% endhighlight %}

Input

M matrix of tensor entries
CS crystal symmetry

Options

rank rank of the tensor
unit physical unit of the entries
name name of the tensor
