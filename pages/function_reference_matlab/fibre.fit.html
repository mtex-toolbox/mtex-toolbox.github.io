determines the fibre that fits best a list of orientations

Syntax

{% highlight matlab %} f = fibre.fit(ori) % fit fibre to a list of orientations {% endhighlight %} {% highlight matlab %} f = fibre.fit(ori,'local') {% endhighlight %} {% highlight matlab %} f = fibre.fit(odf) % fit fibre to a list of orientations {% endhighlight %}

Input

ori1, ori2, ori orientation
odf SO3Fun

Output

f fibre
lambda eigenvalues of the orientation tensor

Options

local fast approach for locally concentrated orientations
