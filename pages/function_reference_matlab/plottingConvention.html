--- title: plottingConvention sidebar: function_reference_sidebar permalink: plottingConvention.html folder: function_reference toc: false --- plottingConvention edit page

class describing the alignment of a reference frame on the screen

Syntax

{% highlight matlab %} % specify a custom plotting convention pC = plottingConvention(outOfScreen,east) plot(ebsd,pC) {% endhighlight %} {% highlight matlab %} % changing the default plotting convention plottingConvention.default.east = yvector {% endhighlight %} {% highlight matlab %} % changing the plotting convention for a dataset % to be used in all future plotting commands ebsd.how2plot = pC {% endhighlight %}

Input

outOfScreen vector3d
east vector3d

Output

pC plottingConvention
