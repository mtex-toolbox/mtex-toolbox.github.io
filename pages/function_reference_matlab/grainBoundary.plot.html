plot grain boundaries

The function plots grain boundaries.

Syntax

{% highlight matlab %} plot(grains.boundary) plot(grains.innerBoundary,'linecolor','r') plot(gB('Forsterite','Forsterite'),gB('Forsterite','Forsterite').misorientation.angle) {% endhighlight %} {% highlight matlab %} % colorize segments according to a list of RGB values plot(gB('Forsterite','Forsterite'),color) {% endhighlight %}

Input

grains grain2d
gB grainBoundary
color n x 3 list of RGB values

Options

linewidth line width
LineColor line color
edgeAlpha (list of) transparency values between 0 and 1
region [xmin xmax ymin ymax] plot only a subregion
DisplayName label to appear in the legend
smooth try to make a smooth connections at the vertices
