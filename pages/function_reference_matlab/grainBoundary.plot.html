plot grain boundaries

The function plots grain boundaries.

Syntax

{% highlight matlab %} plot(grains.boundary) plot(grains.subBoundary,'linecolor','r') {% endhighlight %} {% highlight matlab %} % colorize segments according to some property plot(gB('Forsterite','Forsterite'),gB('Forsterite','Forsterite').misorientation.angle) {% endhighlight %} {% highlight matlab %} % colorize segments according to a list of RGB values plot(gB('Forsterite','Forsterite'),colorList) {% endhighlight %}

Input

grains grain2d
gB grainBoundary
colorList n x 3 list of RGB values

Options

linewidth line width
linecolor line color
edgeAlpha (list of) transparency values between 0 and 1
region [xmin xmax ymin ymax] plot only a subregion
displayName label to appear in the legend
smooth try to make a smooth connections at the vertices
