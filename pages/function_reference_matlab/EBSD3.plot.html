spatial EBSD plot

Syntax

{% highlight matlab %} % colorize according to phase plot(ebsd) {% endhighlight %} {% highlight matlab %} % colorize according to arbitrary value - here MAD plot(ebsd,ebsd.mad) {% endhighlight %} {% highlight matlab %} % colorize according to orientation plot(ebsd('phaseName'),ebsd('phaseName').orientation) {% endhighlight %} {% highlight matlab %} % colorize according to custom color oM = ipfColorKey(ebsd('phaseName')) color = oM.orientation2color(ebsd('phaseName').orientations); plot(ebsd('phaseName'),color) {% endhighlight %} {% highlight matlab %} % specify the color directly and show in Legend badMAD = ebsd.mad > 1; plot(ebsd(badMAD),'faceColor','black,'DisplayName','bad values') {% endhighlight %} {% highlight matlab %} % plot a subregion plot(ebsd,ebsd.orientation,'region',[xmin, xmax, ymin, ymax]) {% endhighlight %}

Input

ebsd EBSD
color length(ebsd) x 3 vector of RGB values

Options

micronbar 'on'/'off'
DisplayName add a legend entry
region [xmin, xmax, ymin, ymax] plotting region

Flags

points plot dots instead of unitcells
exact plot exact unitcells, even for large maps
