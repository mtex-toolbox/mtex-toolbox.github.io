minimum of two pole figures or the minimum of a single polefigure

Syntax

{% highlight matlab %} m = min(pf) pf = min(pf1,pf2) pf = min(pf1,intensity) pf = min(intensity,pf1) {% endhighlight %}

Input

pf PoleFigure
pf1, pf2 PoleFigure
x double

Output

m double
pf PoleFigure
