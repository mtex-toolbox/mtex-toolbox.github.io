The class stressTensor describes a (list) of stress tensors.

Syntax

{% highlight matlab %} % specify stress tensor by matrix M = [1 0 0; 0 0 0; 0 0 0]; sigma = stressTensor(M) {% endhighlight %} {% highlight matlab %} % uniaxial stress in direction v sigma = stressTensor.uniaxial(v) {% endhighlight %}

Input

M 3x3 symmetric matrix
v vector3d

Output

sigma stressTensor
