defines a Bingham distributed ODF

Description

BinghamODF defines a Bingham distributed ODF with A and Lambda.

Syntax

{% highlight matlab %} odf = BinghamODF(kappa,ori) odf = BinghamODF(kappa,f) odf = BinghamODF(kappa,A,CS,SS) odf = BinghamODF(kappa,q,CS,SS) {% endhighlight %}

Input

kappa form parameter
ori principle orientation
f @fiber
A orthogonal 4x4 matrix
q quaternion
CS, SS crystal, specimen symmetry

Output

odf SO3Fun
