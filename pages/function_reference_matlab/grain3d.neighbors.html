returns the Ids of neighboring grains

Syntax

{% highlight matlab %} % neighboring relationships within grains pairs = neighbors(grains) pairs = neighbors(grains1,'matrix') {% endhighlight %} {% highlight matlab %} % all neighbouring relationships pairs = neighbors(grains,'full') {% endhighlight %}

Input

grains grain3d

Output

pairs index list of size N x 2

Flags

matrix get adjacency matrix
