define a equispaced grid localized to a center orientation

Syntax

{% highlight matlab %} ori = localOrientationGrid(CS,SS,maxAngle,'center',center) {% endhighlight %}

Input

CS symmetry
SS symmetry
maxAngle radius of the ball of orientations
center orientation

Output

ori orientation
