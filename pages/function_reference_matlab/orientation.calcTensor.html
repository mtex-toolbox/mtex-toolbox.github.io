compute the average tensor for a vector of orientations

Syntax

{% highlight matlab %} %returns the Voigt--, Reuss-- and Hill-- average of T [TVoigt, TReuss, THill] = calcTensor(ori,T,'weights',w) {% endhighlight %} {% highlight matlab %} % returns the specified , i.e. 'Hill' in this case THill = calcTensor(ori,T,'Hill') {% endhighlight %} {% highlight matlab %} % uses geometric mean instead of arithmetric one TGeom = calcTensor(ori,T,'geometric') {% endhighlight %}

Input

ori orientation
T tensor
w weights for each orientation

Output

TVoigt, TReuss, THill tensor

Options

Voigt Voigt mean
Reuss Reuss mean
Hill Hill mean
geometric geometric mean
