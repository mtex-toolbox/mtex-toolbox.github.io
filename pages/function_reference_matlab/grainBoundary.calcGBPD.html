compute the grain boundary plane distribution

Syntax

{% highlight matlab %} GBPD = calcGBND(gB,ebsd) GBPD = calcGBND(gB,grains) {% endhighlight %} {% highlight matlab %} % use a specific halfwidth GBPD = calcGBND(gB,ebsd,'halfwidth',10*degree) {% endhighlight %}

Input

gB grainBoundary
ebsd EBSD
grains grain2d

Output

gpnd S2FunHarmonic
