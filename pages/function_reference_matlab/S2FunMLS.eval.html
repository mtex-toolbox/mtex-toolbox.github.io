evaluate sF on v via moving least squares (MLS) approximation provide the possibility of also returning the condition numbers of the gram matrices

Syntax

{% highlight matlab %} vals = sF.eval(v) vals = eval(sF,v) {% endhighlight %}

Input

sF the function we want to approximate
v the points where we want to evaluate the MLS approximation

Output

vals the values of sF on v
