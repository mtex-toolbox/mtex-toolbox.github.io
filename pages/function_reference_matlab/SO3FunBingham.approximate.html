Approximate BinghamODF from a given SO3Fun via kernel density estimation

Syntax

{% highlight matlab %} SO3F = SO3FunBingham.approximate(odf) SO3F = SO3FunBingham.approximate(odf,'resolution',5*degree) SO3F = SO3FunBingham.approximate(odf,'SO3Grid',S3G) {% endhighlight %}

Input

odf SO3Fun
S3G rotation

Output

SO3F SO3FunBingham

Options

resolution resolution of the grid nodes of the SO3Grid
SO3Grid grid nodes of the SO3Grid
