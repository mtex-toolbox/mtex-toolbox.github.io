select EBSD data by spatial coordinates

Syntax

{% highlight matlab %} map = findByLocation( ebsd, xy, y ) {% endhighlight %}

Input

ebsd EBSD
xy list of [x(:) y(:)] coordinates, respectively [x(:) y(:) z(:)]

Output

ebsd EBSD subset

Example

{% highlight matlab %} mtexdata forsterite silent {% endhighlight %} {% highlight matlab %} plot(ebsd) p = [10000 5000] %ginput(1) g = findByLocation(ebsd,p) {% endhighlight %} {% highlight plaintext %} p = 10000 5000 g = 73401 {% endhighlight %}
