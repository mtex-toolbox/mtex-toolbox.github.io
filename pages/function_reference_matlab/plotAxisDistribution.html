plot axis distribution

Syntax

{% highlight matlab %} plotAxisDistribution(cs) % random axis distribution plotAxisDistribution(cs1,cs2) % random misorientation axis distribution plotAxisDistribution(mori) % axes in crystal coordinates plotAxisDistribution(mori,cs1) % axes in crystal coordinates plotAxisDistribution(ori1,ori2) % axes in specimen coordinates plotAxisDistribution(odf) {% endhighlight %}

Input

odf ODF
mori @misorientation
ori1,ori2 orientation
cs1, cs2 crystalSymmetry

Options

RESOLUTION resolution of the plots

Flags

antipodal include antipodal symmetry
complete plot entire (hemi)--sphere
noLabel do not annotate
