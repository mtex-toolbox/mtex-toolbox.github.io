--- title: steepestDescent sidebar: function_reference_sidebar permalink: S2FunHarmonic.steepestDescent.html folder: function_reference toc: false --- steepestDescent edit page

calculates the minimum of a spherical harmonic

Syntax

{% highlight matlab %} [pos, v] = steepestDescent(sF) % the position where the minimum is attained {% endhighlight %} {% highlight matlab %} [pos, v] = steepestDescent(sF,'numLocal',5) % the 5 largest local minima {% endhighlight %} {% highlight matlab %} % with all options [pos, v] = steepestDescent(sF, 'startingnodes') {% endhighlight %}

Output

v double
pos vector3d

Options

iterMax number of iterations
numLocal number of peaks to return
startingNodes vector3d
tolerance minimum distance between two peaks
resolution minimum step size
maxStepSize maximum step size
