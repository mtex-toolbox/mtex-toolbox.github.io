returns

Syntax

{% highlight matlab %} [vote, numVotes] = majorityVote(idList,voteList) {% endhighlight %}

Input

idList list of ids
voteList list of votes

Output

vote most frequent vote in voteList(idList == id) for each id
numVotes number occurences of vote in voteList(idList == id) for each id

Options

strict only assign a vote if all votes coincide, otherwise set in to nan

Example

{% highlight matlab %} idList = [1 1 1 1 1 2 3 2 3 2 3]; voteList = [5 9 5 9 9 4 1 4 2 4 1]; {% endhighlight %} {% highlight matlab %} majorityVote(idList,voteList) {% endhighlight %} {% highlight plaintext %} ans = 9 4 1 {% endhighlight %} {% highlight matlab %} majorityVote(idList,voteList,'strict') {% endhighlight %} {% highlight plaintext %} ans = NaN 4 NaN {% endhighlight %}
