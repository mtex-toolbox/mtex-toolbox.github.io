calculates the gradient of a spherical harmonic

Syntax

{% highlight matlab %} sVF = grad(sF) % returns the gradient as a spherical vector field g = grad(sF, v) % return the gradient in point v as vector3d {% endhighlight %}

Input

sF S2FunHarmonic
v vector3d

Output

sVF @sphericalVectorFieldHarmonic
g vector3d
