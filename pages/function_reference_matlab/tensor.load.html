import Tensor data

Description

tensor.load is a high level method for importing tensors from external files. It autodetects the format of the file.

Syntax

{% highlight matlab %} T = tensor.load(fname,cs) {% endhighlight %}

Input

fname filename
cs crystalSymmetry

Output

T tensor
