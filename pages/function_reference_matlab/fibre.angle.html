angle fiber to orientation or fiber to fibre

Syntax

{% highlight matlab %} omega = angle(f,ori) % angle orientation to fibre [omega, oriF] = angle(f,ori) {% endhighlight %} {% highlight matlab %} omega = angle(f1,f2) % angle fibre to fibre {% endhighlight %}

Input

f, f1, f2 fibre
ori orientation

Output

omega double
oriF orientation on the fibre that minimizes the angle
