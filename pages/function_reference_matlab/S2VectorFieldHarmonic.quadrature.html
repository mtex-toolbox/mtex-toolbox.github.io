Compute the S2-Fourier/harmonic coefficients of an given S2VectorField or given evaluations on a specific quadrature grid.

Syntax

{% highlight matlab %} sVF = S2VectorField.quadrature(v, value) sVF = S2VectorField.quadrature(f) sVF = S2VectorField.quadrature(f, 'bandwidth', bw) {% endhighlight %}

Input

value vector3d
v vector3d
f S2VectorField, @function_handle in vector3d

Output

sVF S2VectorFieldHarmonic

Options

bw degree of the spherical harmonic (default: 128)
