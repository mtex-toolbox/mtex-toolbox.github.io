Syntax

{% highlight matlab %} rgb = spectralTransmission(rI,vprop,p,thickness) {% endhighlight %} {% highlight matlab %} rgb = spectralTransmission(rI,vprop,p,thickness,'polarizationDirection',p) {% endhighlight %} {% highlight matlab %} rgb = spectralTransmission(rI,vprop,p,thickness,'phi',phi) {% endhighlight %} {% highlight matlab %} plot(rI.spectralTransmission(thickness)) {% endhighlight %}

Input

rI refractiveIndexTensor
vprop propagation direction
thickness
p direction of the polarizer
tau angle between polarizer and analyzer

Example

{% highlight matlab %} thickness = 20000; rI = refractiveIndexTensor.calcite plot(rI.spectralTransmission(thickness),'rgb','3d') {% endhighlight %} {% highlight plaintext %} rI = refractiveIndexTensor (Calcite) rank: 2 (3 x 3) 1.66 0 0 0 1.66 0 0 0 1.486 {% endhighlight %}
