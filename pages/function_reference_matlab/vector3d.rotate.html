rotate vector3d by rotation or orientation

Description

Either |v| or |rot| are single elements or both have the same size. The
ouptut |v| will have the same size as the biger of both input arrays.

Syntax

{% highlight matlab %} v = rotate(v,20*degree) % rotation about the z-axis rot = rotation.byEuler(10*degree,20*degree,30*degree) v = rotate(v,rot) {% endhighlight %}

Input

v vector3d
q quaternion

Output

r q * v
