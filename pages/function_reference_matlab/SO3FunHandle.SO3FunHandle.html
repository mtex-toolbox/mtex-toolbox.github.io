a class representing a function on the rotation group by an function handle

Syntax

{% highlight matlab %} SO3F = SO3FunHandle(fun) {% endhighlight %}

Input

fun @function_handle

Output

SO3F SO3FunHandle

Example

{% highlight matlab %} r = orientation.rand; SO3F = SO3FunHandle(@(rot) angle(rot,r)) {% endhighlight %} {% highlight plaintext %} SO3F = SO3FunHandle (y↑→x → y↑→x) eval: @(rot)angle(rot,r) {% endhighlight %}
