represent orientations in a gridded structure to allow quick access

Syntax

{% highlight matlab %} S3G = homochoricSO3Grid(CS) S3G = homochoricSO3Grid(CS,SS,'resolution',2.5*degree) % specify the resolution {% endhighlight %} {% highlight matlab %} % fill only a ball with radius of 20 degree S3G = equispacedSO3Grid(CS,'maxAngle',20*degree) {% endhighlight %}

Input

CS crystalSymmetry
SS specimenSymmetry
res resolution in radiant

Output

S3G homochoricSO3Grid

Options

maxAngle radius of the ball to be filled
no difference is made between antipodal quaternions
