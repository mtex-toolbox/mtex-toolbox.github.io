factorizes s1 and s2 into rot1, d and rot2 such that s1 = rot1 * d and s2 = d * rot2

Syntax

{% highlight matlab %} [rot1,d,rot2] = factor(s1,s2) {% endhighlight %}

Input

s1, s2 symmetry

Output

rot1, rot2, d rotation
