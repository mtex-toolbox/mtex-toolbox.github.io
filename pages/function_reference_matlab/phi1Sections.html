phi1 sections for ODF and orientation plotting

Syntax

{% highlight matlab %} oS = phi1Sections(cs1,cs2,'sections',5) oS = phi1Sections(cs1,cs2,'phi1',(0:15:90)*degree) {% endhighlight %}

Input

cs1, cs2 crystalSymmetry, specimenSymmetry

Options

sections number of sections
phi1 explicit section values
