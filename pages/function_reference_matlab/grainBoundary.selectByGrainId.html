select grain boundaries by pairs of grainId

Syntax

{% highlight matlab %} [gBSub, pairId, ind] = selectByGrainId(gB,grainId) {% endhighlight %}

Input

gB grainBoundary
grainId matrix of pairs of grainIds

Output

gBSub grainBoundary
pairId gBSub.grainId = grainId(pairId,:)
ind gBSub = gB(ind)
