maximum of two pole figures or the maximum of a single polefigure

Syntax

{% highlight matlab %} m = max(pf) pf = max(pf1,pf2) pf = max(pf1,x) = max(x,pf1) {% endhighlight %}

Input

pf PoleFigure
pf1, pf2 PoleFigure
x double

Output

m double
pf PoleFigure
