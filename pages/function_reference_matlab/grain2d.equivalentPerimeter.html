returns the equivalent perimeter of grain-polygon

Description

The equivalent perimeter of grain-polygon is defined as

\[ p = 2 \pi ER \],

where \(ER\) is the equivalent radius of a grain

Syntax

{% highlight matlab %} p = equivalentPerimeter(grains) {% endhighlight %}

Input

grains grain2d

Output

p equivalent perimeter in measurement units
