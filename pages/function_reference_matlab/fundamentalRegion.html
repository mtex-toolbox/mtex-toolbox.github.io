get the fundamental zone in orientation space for a symmetry

Syntax

{% highlight matlab %} oR = fundamentalRegion(cs) oR = fundamentalRegion(cs1,cs2) [oR,dcs,nSym] = fundamentalRegion(ori) [oR,dcs,nSym] = fundamentalRegion(odf) {% endhighlight %}

Input

cs,cs1,cs2 symmetry
ori orientation
odf SO3Fun

Output

oR orientationRegion
dcs common symmetries in cs1 and cs2
nSym number of zones

Options

invSymmetry wheter mori == inv(mori)
