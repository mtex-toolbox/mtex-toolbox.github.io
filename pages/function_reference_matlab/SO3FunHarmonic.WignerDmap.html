create harmonic Representation of the normalized Wigner-D function with harmonic degree N and indices k,l : \(D^n_{k,l}\)

Syntax

{% highlight matlab %} SO3F = SO3FunHarmonic.WignerDmap(N,k,l) SO3F = SO3FunHarmonic.WignerDmap(N,'full') {% endhighlight %}

Input

N harmonic degree of Wigner-D function
k,l indices of Wigner-D function with harmonic degree N

Output

SO3F SO3FunHarmonic

Options

'full' creates f(x) = sum_{k,l} D_{k,l}^N (x), i.e. sum of all
normalized Wigner D functions with harmonic degree N
