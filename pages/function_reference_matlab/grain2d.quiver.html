plot directions at grain centers

Syntax

{% highlight matlab %} quiver(grains,dir,'color','r') {% endhighlight %} {% highlight matlab %} mtexdata forsterite grains = calcGrains(ebsd('indexed')) quiver(grains,grains.meanRotation.axis,'color','r') {% endhighlight %}

Input

grains grain2d
dir vector3d

Options

antipodal plot directions or axes
maxHeadSize size of the arrow
project2Plane project direction into plane
