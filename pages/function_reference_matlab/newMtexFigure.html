set up a new plotting figure

Syntax

{% highlight matlab %} newMtexFigure('figSize','large') newMtexFigure('layout',[2,3]) {% endhighlight %}

Options

figSize huge, large, normal, small, tiny
layout [numberRows, numberColumns]
