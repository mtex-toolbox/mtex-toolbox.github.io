interpolate at arbitrary points (x,y)

Syntax

{% highlight matlab %} ebsdNew = interp(ebsd,xNew,yNew) {% endhighlight %} {% highlight matlab %} ebsdNew = interp(ebsd,xNew,yNew,'method','invDist') {% endhighlight %}

Input

ebsd EBSDsquare
xNew, yNew new x,y coordinates

Output

ebsdNew EBSD with coordinates (xNew,yNew)

Options

method 'invDist', 'nearest'
