remove vectors v2 from a set of vectors v1

Syntax

{% highlight matlab %} v = setdiff(v1,v2) v = setdiff(v1,v2,'antipodal') {% endhighlight %}

Input

v1, v2 vector3d

Output

v vector3d

Options

antipodal include antipodal symmetry
