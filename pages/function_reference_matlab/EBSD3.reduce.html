reduce ebsd data by a factor

Syntax

{% highlight matlab %} ebsd = reduce(ebsd) % take every second pixel horiz. and vert. ebsd = reduce(ebsd,3) % take every third pixel horiz. and vert. {% endhighlight %}

Input

ebsd EBSD
factor resample ebsd at rate factor (integer)

Output

ebsd EBSD
