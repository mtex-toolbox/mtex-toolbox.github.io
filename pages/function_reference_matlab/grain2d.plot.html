colorize grains

Syntax

{% highlight matlab %} plot(grains) % colorize by phase plot(grains,property) % colorize by property plot(grains,cS) % visualize crystal shape plot(grains,S2F) % visualize a tensorial property on top of the grains {% endhighlight %}

Input

grains grain2d
cS crystalShape

Output

h graphics handle
mP mapPlot

Options

PatchProperty all matlab patch properties
noBoundary do not plot boundaries
displayName name used in legend
region [xmin, xmax, ymin, ymax] of the plotting region
scale scaling of crystal shapes and tensorial properties (0.3)
micronBar 'on' / 'off'
