flip a boundary from phase1/phase2 to phase2/phase1

Syntax

{% highlight matlab %} gB = flip(gB) % flip all boundaries gB = flip(gB,ind) % flip the boundaries specified in ind {% endhighlight %}

Input

gB grainBoundary
ind indeces of the boundaries to flip

Output

grains grainBoundary
