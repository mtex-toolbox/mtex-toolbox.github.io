--- title: SO3Fun.ensureCompatibleSymmetries sidebar: function_reference_sidebar permalink: SO3Fun.ensureCompatibleSymmetries.html folder: function_reference toc: false --- SO3Fun.ensureCompatibleSymmetries edit page

For calculating with SO3Fun (+, -, .*, ./, conv, ...) we have to verify that the symmetries are suitable.

By default the Left and Right symmetries of both functions have to coincide. By convolution of SO3Fun's the Left symmetry of one function have to coincide with the right symmetry of the other function.

Evaluation of some SO3Fun in some orientation also needs suitable symmetries.

Syntax

{% highlight matlab %} ensureCompatibleSymmetries(SO3F1,SO3F2) ensureCompatibleSymmetries(SO3F1,sF) ensureCompatibleSymmetries(SO3F1,ori) ensureCompatibleSymmetries(SO3F1,SO3F2,'conv') ensureCompatibleSymmetries(SO3F1,'antipodal') {% endhighlight %}

Input

SO3F1, SO3F2 SO3Fun
sF S2FunHarmonicSym
ori orientation

Output

msg yields a error message if the symmetry do not match

Options

conv be shure switched symmetries match
