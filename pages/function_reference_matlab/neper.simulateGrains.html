generating 3d neper tessellations

Syntax

{% highlight matlab %} neper.init neper.filepath='C:\Users\user\Work\Mtex\NeperExamples' %select working folder, default: @tempdir numGrains = 100; odf = unimodalODF(orientation.rand); grains = neper.simulateGrains(numGrains, odf) {% endhighlight %} {% highlight matlab %} ori = discreteSample(odf,numGrains) grains = neper.simulateGrains(ori,'silent') {% endhighlight %}

Input

numGrains number of grains
odf SO3Fun
ori orientation
'silent' print log file, no console output

Output

grains grain3d
