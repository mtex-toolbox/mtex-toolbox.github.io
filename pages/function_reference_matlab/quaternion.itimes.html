computes inv(o1) .* o2

Syntax

{% highlight matlab %} q = q1 .* q2 {% endhighlight %} {% highlight matlab %} q = times(q1,q2) q = itimes(q1,q2,1) % inv(o1) .* o2 q = itimes(q1,q2,0) % o1 .* inv(o2) {% endhighlight %}

Input

q1 quaternion
q2 quaternion
takeRight logical, use as output left or right input

Output

q quaternion
