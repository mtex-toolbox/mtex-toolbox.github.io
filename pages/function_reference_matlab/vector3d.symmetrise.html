symmetrically equivalent directions and its multiple

Syntax

{% highlight matlab %} vSym = symmetrise(v,S) {% endhighlight %} {% highlight matlab %} % include antipodal symmetry vSym = symmetrise(v,S,'antipodal') {% endhighlight %} {% highlight matlab %} % exclude antipodal symmetry vSym = symmetrise(v,S,'noAntipodal') {% endhighlight %} {% highlight matlab %} % every symmetrically equivalent direction only once [vSym,l,sym] = symmetrise(v,S,'unique') {% endhighlight %} {% highlight matlab %} % every symmetrically equivalent axis only once [vSym,l,sym] = symmetrise(v,S,'unique','noAntipodal') {% endhighlight %}

Input

v vector3d
S symmetry

Output

vSym S * v vector3d
l multiplicity of the crystal directions
sym rotation

Flags

antipodal include antipodal symmetry
noAntipodal do not include antipodal symmetry (without option unique)
noAntipodal do not remove antipodal vectors (with option unique)
unique only return distinct axes or directions (noAntipodal)
