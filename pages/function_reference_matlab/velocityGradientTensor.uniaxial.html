defines uniaxial tension/compression tensors

Syntax

{% highlight matlab %} % axi symmetric uniaxial tension L = r .* velocityGradientTensor.uniaxial(d,q) {% endhighlight %} {% highlight matlab %} % uniaxial compression L = -velocityGradientTensor.uniaxial(d,q) {% endhighlight %}

Input

d tension direction vector3d
q [0 .. 1] non axial symmetric portion (0.5 is symmetric and default)
r strain rate

Output

L velocityGradientTensor
