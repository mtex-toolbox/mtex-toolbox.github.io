compute the maximal distance of a point on the meshNorm of a spherical grid

The mesh norm \(\delta\) ist the maximum of the minimal distance \(d(x,y)\) of any point \(y\) on the sphere to any point \(x\) of the grid \(G\).

\[ delta = \max_{x \in S^2} \min_{y \in G} d(x,y)\]

Syntax

{% highlight matlab %} delta = meshNorm(G) {% endhighlight %}

Input

G spherical grid vector3d

Output

delta mesh norm
