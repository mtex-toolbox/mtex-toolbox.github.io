find ind such that grains.id(ind) == id

Syntax

{% highlight matlab %} ind = id2ind(grains,id) {% endhighlight %}

Input

grains grain2d
id a list of id's as stored in grains.id

Output

ind a list indices such that grains.id(ind) == id
