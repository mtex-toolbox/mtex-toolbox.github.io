Young's modulus for an elasticity tensor

Description

\[E = \frac{1}{S_{ijkl} x_i x_j x_k x_l}\]

Syntax

{% highlight matlab %} E = YoungsModulus(C) E = YoungsModulus(C,x) {% endhighlight %}

Input

C elastic stiffness tensor
x list of vector3d

Output

E Youngs modulus
