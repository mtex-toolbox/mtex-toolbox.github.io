Extract the corresponding S2FunHarmonic of an SHT file. examples of SHT files can be found at https://github.com/EMsoft-org/SHTdatabase/tree/master/EBSD

Syntax

{% highlight matlab %} sF = SHTextractor(file) {% endhighlight %}

Input

file path/name of SHT-File

Output

sF S2FunHarmonic
