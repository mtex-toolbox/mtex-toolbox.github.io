generates a patch object to be included into a grain or EBSD plot

Syntax

{% highlight matlab %} p = axialSymbol(center,v,symbolSize) {% endhighlight %}

Input

center [x,y] of symbol center
v axial direction
symbolSize size of the symbol

Output

p patch object
