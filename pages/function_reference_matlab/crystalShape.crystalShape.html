a class representing crystal shapes.

The code of this class is based on the paper

Enderlein, J., 1997. A package for displaying crystal morphology. Mathematica Journal, 7(1).

Syntax

{% highlight matlab %} cS = crystalShape(N,habitus,extension) {% endhighlight %}

Example

{% highlight matlab %} mtexdata titanium; cS = crystalShape.hex(ebsd.CS) close all plot(cS) {% endhighlight %} {% highlight plaintext %} ebsd = EBSD (y↑→x) Phase Orientations Mineral Color Symmetry Crystal reference frame 0 8100 (100%) Titanium (Alpha) LightSkyBlue 622 X||a, Y||b*, Z||c* Properties: ci, grainid, iq, sem_signal Scan unit : um X x Y x Z : [0, 996] x [0, 998] x [0, 0] Normal vector: (0,0,1) cS = crystalShape mineral: Titanium (Alpha) (622, X||a, Y||b*, Z||c*) vertices: 12 faces: 8 {% endhighlight %}
