projects orientations to a fundamental region

Syntax

{% highlight matlab %} [q,omega] = getFundamentalRegion(o,q_ref) {% endhighlight %}

Input

o orientation
q_ref reference quaternion

Output

q quaternion
omega rotational angle to reference orientation
