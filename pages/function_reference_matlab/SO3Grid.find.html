return indece and distance of all nodes within a eps neighborhood

Syntax

{% highlight matlab %} [ind,dist] = find(SO3G,nodes,radius) {% endhighlight %}

Input

SO3G SO3Grid
nodes quaternion
radius double

Output

ind index of the closes grid point
dist misorientation angle
