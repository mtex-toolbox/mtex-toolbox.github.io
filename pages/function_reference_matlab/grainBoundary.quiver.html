plot directions at grain boundaries

Syntax

{% highlight matlab %} quiver(gB,gB.direction,'linecolor','r') {% endhighlight %}

Input

gB grainBoundary
dir vector3d

Example

{% highlight matlab %} mtexdata forsterite silent grains = calcGrains(ebsd('indexed')); plot(grains(1437)) hold on quiver(grains(1437).boundary,grains(1437).boundary.calcMeanDirection,'color','r') hold off {% endhighlight %}
