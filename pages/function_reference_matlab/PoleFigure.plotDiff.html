Syntax

{% highlight matlab %} plotDiff(pf,pf2) plotDiff(pf,odf) {% endhighlight %}

Input

pf PoleFigure

odf SO3Fun

Options

RP calculate RP error
l1 calculate l^1 error
l2 calculate mean square error
