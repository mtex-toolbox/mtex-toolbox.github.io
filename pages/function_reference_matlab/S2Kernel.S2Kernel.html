The class S2Kernel is needed in MTEX to define the specific form of fibre ODFs. It has to be passed as an argument when calling the methods fibreODF. For more information take a look at the documentation.

Syntax

{% highlight matlab %} S2K = S2Kernel(A, varargin) {% endhighlight %}
