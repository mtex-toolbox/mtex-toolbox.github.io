slice 3d-ebsd to obtain 2d-ebsd

Syntax

{% highlight matlab %} N = vector3d(1,1,1) % plane normal P0 = vector3d(0.5, 0.5, 0.5) % point within plane ebsd = ebsd3.slice(N,P0) {% endhighlight %} {% highlight matlab %} plane = plane3d.fit(v) ebsd = ebsd3.slice(plane) {% endhighlight %}

Input

ebsd3 EBSD3
plane plane3d

Output

ebsd EBSD
