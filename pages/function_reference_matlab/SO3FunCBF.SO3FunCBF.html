a class representing fibre symmetric ODFs

Syntax

{% highlight matlab %} SO3F = SO3FunCBF(h,r,weights,psi) SO3F = SO3FunCBF(fibre,'halfwidth',10*degree) {% endhighlight %}

Input

h vector3d
r vector3d
weights double
psi S2Kernel
fibre fibre

Output

SO3F SO3FunCBF

Example

{% highlight matlab %} cs = crystalSymmetry.load("Mg-Magnesium.cif"); fibre = fibre.beta(cs); SO3F = SO3FunCBF(fibre,'halfwidth',10*degree) {% endhighlight %} {% highlight plaintext %} Warning: Miller indece are not orthogonal. Maximum deviation is 25 degree SO3F = SO3FunCBF (Mg → y↑→x) kernel: de la Vallee Poussin, halfwidth 10° fibre : (11 7 -18 12) || 0.16361,0.13413,0.97736 weight: 1 {% endhighlight %}
