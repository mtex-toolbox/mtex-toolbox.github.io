implements e .* c and c .* e

Description

If e is a matrix of embeddings and c is a matrix of coefficients then e .* c is again a matrix of embeddings defined by

\[ [\mathrm{e .* c}]_{j\ell} = mathrm{e}_{j\ell} \mathrm{c}_{j \ell}\]

Syntax

{% highlight matlab %} out = e .* c out = c .* e out = e1 .* e2 {% endhighlight %}

Input

e, e1, e2 embedding
c double

Output

out embedding
