The class SO3Kernel is needed in MTEX to define the specific form of unimodal ODFs. It has to be passed as an argument when calling the methods uniformODF. For more information take a look at the documentation.

Syntax

{% highlight matlab %} psi = SO3Kernel(A,varargin) {% endhighlight %}
