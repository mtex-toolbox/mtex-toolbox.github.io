check for zero regions in experiementl pole figures

Syntax

{% highlight matlab %} isZero = checkZeroRange(zrm,ori) % check at orientation in all pole figures isZero = checkZeroRange(zrm,v,i) % check at position in the i-th pole figure {% endhighlight %}

Input

zrm zeroRangeMethod
ori orientation
v vector3d
i check in the i~th pole figure

Output

isZero boolean vector of length(ori)
