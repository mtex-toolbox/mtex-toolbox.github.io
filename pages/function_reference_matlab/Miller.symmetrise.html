symmetrcially equivalent directions and its multiple

Syntax

{% highlight matlab %} mSym = symmetrise(m) {% endhighlight %} {% highlight matlab %} % include antipodal symmetry mSym = symmetrise(m,'antipodal') {% endhighlight %} {% highlight matlab %} % exclude antipodal symmetry mSym = symmetrise(m,'noAntipodal') {% endhighlight %} {% highlight matlab %} % every symmetrically equivalent direction only once [mSym,l,sym] = symmetrise(m,'unique') {% endhighlight %} {% highlight matlab %} % every symmetrically equivalent axis only once [mSym,l,sym] = symmetrise(v,S,'unique','noAntipodal') {% endhighlight %}

Input

v Miller

Output

mSym sym * m Miller
l multiplicity of the crystal directions
sym rotation

Flags

antipodal include antipodal symmetry
noAntipodal do not include antipodal symmetry (without option unique)
noAntipodal do not remove antipodal vectors (with option unique)
unique only return distinct axes or directions (noAntipodal)
